#4

上午简单测试了一下，使用的是我提供的GOG版本，随便找了个XTC2.0的包（无2.2patch）打上就直接变成XTC了

我估计大概率是你找错安装位置了，XTC的安装器上有一行字，choose the folder you wish to install XTC to.(It should be the same folder that X3TC.exe is located in.)

steam版不知道，如果你使用GOG版的话，安装路径应该是.../X3 Terran War Pack，目录里会有个X3TC.exe文件————同目录里还有X3AP和X3FL，不用管他们

安装好后在此目录中打开X3TC.exe进行游戏，不要使用Launch X3 - Terran War Pack这个快捷方式，因为快捷方式指向路径可能是错的

你先试一下，有问题继续反馈。我建议你先别打升级和汉化，先试一下XTC的包能不能打成功，成功了后再去打别的，这样debug比较省事一些

PS：不排除XTC安装被被杀软拦截，因为这种自解压的7zip包很容易被误杀
