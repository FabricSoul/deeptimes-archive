Ship Rebalance Mod: Continued v0.90a

Latest Release: 14/12/10

(updated cockpits 21/10/10; trails 10/12/10; Shipyards 14/12/10; SSRP 12/10/10; Jobs 10/12/10; Hulls 10/12/10; Weapons Dealers 14/12/10)

-----------------------------------------------------------------

The SRM rebalances almost all ships in the game with a view to making every ship unique and useful. We have tried to ensure there are no "I Win" ships and that all ships within a class are balanced across the races. All ship stats are taken into account from speed and acceleration to cargo and price. It also adds many optional addon packs to further enhance your game and is compatible with many other popular mods. Please see the compatibility section below as many mods require special installation instructions to work with the SRM.

As an added bonus the SRM also adds many (well over 80!) new ships into the game taken from some of the excellent ship mods available, as well as some SRM exclusive ships.

------------------------------------------------------------------

This is a continuation of the Ship Rebalance Mod by SomeoneElse (original thread). It contains all the changes made in the original and is compatible with old SRM saves.

The SRM is fully compatible with X3TC v3.0

Much of the rebalancing has been done with the Combat Mod 4 in mind so I highly recommend this for use with the SRM - although it is by no means essential.

简单说说要点：这个SRMC mod（Ship Rebalance Mod: Continued姑且由我这么简化了吧）是原来那个SRM mod的一个进化版（原先的应该是SRM0.59，下载时注意区分），其字面意思就是：舰船平衡mod：拓展版。顾名思义，这个mod旨在对各类舰船进行平衡从而促进其实用性和独特性。从速度、加速度到货仓、价格，这个mod都进行了适当的平衡。不仅如此，这个mod还引入了许多现有的舰船mod，从而在游戏中加入了超过80种新船。

由于很多平衡是结合Combat Mod 4进行的所以推荐大家也把这个装上。

另外这个mod和原版SRM存档兼容，和大多数mod及脚本兼容（The Combat Mod 4, Pirate Guild, Military Base Response Revamp, Improved Races, Race Responce Fleets, Khaak Invasions, NPC Bailing Addon, MARS, Amurph0's Shipyards, Missile Defense Mk 2, Missile Rebalance Mod, Medusa Missile system, Advanced Jumpdrive. ），但是安装顺序要注意，需要装n个mod的大神请去E文论坛自己研究，出现任何问题本人概不负责~

另外特别提醒如果在原版基础上装此mod请先跳到一个没有船的空星区~否则很可能在安装后第一次载入存档时会over~
