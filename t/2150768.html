2.5 beta 机翻内容~

原作者：月下小憩　发表：2019-4-14　来源：bbs.deeptimes.net（已关站）· Wayback 存档

#1

新特点:补给舰现在可用，航母也可以补给他们的战斗机。

新功能:扇区黑名单设置，防止船舶进入某些区域。

新功能:库存管理与锁盒在空间和库存存储在玩家总部。

新特性:Tobii眼球跟踪支持(只在这个版本中支持windows, Linux支持即将到来)。

在百科全书中添加了关于恒星系统和行星的图片和背景信息。

增加了NPC船只在玩家造船厂建造、补给或修理的航海日志条目。

添加注意/提示时，鼠标直接转向模式是活跃的。

增加了更多的变化建设站，救援船和扫描任务。

在玩家信息菜单中添加了新的全局订单部分。

增加了选项的货物转移和贸易与建设存储到停靠菜单。

增加了玩家拥有的船只的库存信息。

向映射中的对象列表和属性添加类别。

添加了按名称、大小或外壳对对象列表和属性所属菜单进行排序的选项。

在贸易菜单中增加了警告，如果船只不能运输货物。

增加了在船在空间站部署之前取消建造船任务的可能性。

将当前构建进度信息添加到地图中站点条目的模块类型摘要行。

增加了配置自动保存间隔的选项(并在手动保存之后立即阻止自动保存)。

在研究菜单中添加关于被黑的研究模块的信息。

在信息菜单和逻辑站概述中添加关于被黑生产模块的信息。

增加了为每个站点单独配置逻辑站点概览图数据的能力。

添加残骸过滤器选项映射。

增加了室内的声音，房间，商店和船舶桥梁。

增加了采矿和补给教程。

改进的选择多种商品的行为和地图过滤器。

改进的扇区外火力计算。

改进的资源预览在船舶建造/升级菜单。

改进的卫星部署教程。

提高了加载和保存性能。

改进了减少玩家库存的界面。

改进了地图对象列表的性能，当看到许多玩家站。

改进了在处理物品短缺时的阵营行为。

改进了飞行模式下的编队行为。

改进控制器支持船舶建造菜单中的初始船舶类型选择。

在某些情况下改进了船舶的进坞和出坞动作。

改善自由飞行警察的行为。

改进了主力舰在战斗中的无人机操控能力。

改进了绘制模式的任务奖励文字。

改进了以站为基础的贸易船和采矿船交易或收获的货物的更新。

改进了逻辑站概览图数据的选择。

改进的人工智能站生成逻辑，使选择大型居住模块更有可能在适当的地方。

从建造空间站任务中移除飞往目标的苍蝇。

移除加载包含比玩家现有更多风险模块的构建计划的能力。

在支付了一个便宜的许可证后，将一个空的车站地块转移到一个拥有昂贵许可证的地点而不产生任何后果的能力被移除。

修正了在建造空间站任务中移动空间站的问题。

修正了营救船的任务可能会造成无人驾驶的船只无法清理。

固定的救援船任务奖励计算在变种中，船必须交付。

固定翘曲到一个不同的部门时，起床在特定阶段的总部任务。

固定的导弹射程计算哑弹。

固定非常慢的任务报价显示，如果许多报价是已知的。

修正了在建船舶信息菜单打开时的菜单错误。

修正了巨大的玩家拥有的站不显示所有存储的商品在信息菜单。

修正百科全书链接到船只不能正常工作。

固定NPC购买/出售订单不履行最低金额。

修正了等待升级的船只。

固定错误消息时，试图解锁空诱导的数据泄漏船上。

固定码头速度限制应用时，旅行模式是积极的。

固定的船舶停靠在移动的物体出现扫射非常快在某些时间。

固定的警用船留在空间，他们不再有警察的权力。

修正了一些派系的警用船不被命名为警察。

修正了属于Teladi交易站的防御无人机在完成任务后无法对接的问题。

固定站可能试图与其他站交易时，他们不再有任何可用的货运无人机。

为入侵而建造的固定船只，有时不能执行预定的命令。

修正了下级在某些情况下对攻击没有反应的问题。

固定的船只攻击目标以外的交战区，当他们的方式，他们的交战区。

修正了中队下属在加入指挥官的途中被敌方目标分散注意力的问题。

修正了当玩家在航母上时，航母的下级在离开航母前等待玩家的允许才发动攻击的问题。

进入 2.5beta 的方法

Make sure that your savegames are backed up!

To start using the Beta version, right-click X4: Foundations in the Steam client

Select Properties and go to the Betas tab

Enter the password IBackedUpMySaves

Select public_beta from the dropdown list.

To return to using the non-Beta version, select NONE from the same dropdown list.
