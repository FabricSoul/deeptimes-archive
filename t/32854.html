我来论坛也两周。我本人上手时上得我抓狂。然后发现碰到上手麻烦的也并非只有我一个人，于是决定发篇新人上手的教程。

这篇帖子并非特意写的，由于自己没有玩过重聚对X3完全无概念，在论坛上狂发问。所以最初是写给我自己看的，是我自己碰到上手麻烦的时候做的记录。后来发现稍微修改下，就可以用来做教程。

这里先给个建议，把游戏升到TC 2.5 版，因为我是从 2.5 版后开始玩的，部分内容可能和 2.5 版以前不一样。再有就是建议玩英文版，因为现在的中文版把名称部分也给翻译了，造成查资料不方便。仅个人建议。。。

好了，废话少说，拿出来给和我差不多的人分享。

因为论坛对单帖有严格的图片限制和字数限制，只好分楼层发。

==========

Talk to Lt.Samuel Plinter

流程：

必须在足够近的距离才能够通话，所以先要靠近 Lt. Plinter ，然后才能和他通话。那艘 Terran Scimitar 就是 Lt. Plinter 的座舰。对话时选“1、I'm reporting in for patrol duty.”（我来负行巡逻职责）接第一个主线任务。

相关操作：

点左边的 Personal -〉My Missions 弹出任务菜单，可以看到你接了些任务，进一步点这个任务，弹出菜单再点 Show Briefing 可以看到任务详细内容。按 F1 切换第一人称视角炮位,和按 F2 是第三人称视角看自己船只，想看其它角度，用小键盘上的数字键。

接下来是飞行。先说手动飞行，X 是加速，Z 是减速，按鼠标左键不放就可以调整船头方向，鼠标右键是开火。Q 和 E 分别是逆时针和顺时针翻转（当你发现地图表现和实际情况相反时，最好回忆一下 Q 和 E 的作用）。

然后说下三种方法可以飞向目标，1、肉眼搜寻后，左键锁定，再按 Shift + A(自动)，或者手动飞过去。2、用雷达搜寻，点屏幕左边的 Navigation（导航）-> Sector Map (区域地图)，然后可以看到地图右侧列出了本区域所有的空间站和舰船。左键点他的座舰，会弹出一个菜单，点 AutoPilot:Follow 则会直接自动飞向这个目标。3、雷达找到目标后，点 Select as target 先锁定这个目标，再按 Shift + A 跟踪。（具体用哪种要根据当时情况。一般说来，目标是无敌意用第2种；目标有敌意但距离很远，用第三种；目标有敌意且距离很近，用第一种）在飞船的飞行过程中可以按“J”激活“Singularity Engine Time Accelerator”（奇点时间加速引擎）缩写 SETA ，可以加快游戏中时间的流逝，节约现实中的时间。如果你以后的飞船没有这项能力，就说明这艘飞船没有安装 SETA ，你需要去买一个。目标足够近后，点目标，弹出菜单后点 Comms 就可以通话了。通话菜单打开后，通常和任务直接有关的对话，都放在最靠前的选项。这些操作是以后经常要干的事情，就不再重复了。

注：

通常情况下在 Sector Map ALL 页面里，除了你自己的财产以绿色标识以外，其它蓝色的部分空间站列表在上方，船只列表会列在空间站列表的下方，太空漂浮物（包括宇航员）会在船只列表的下方。如果出现了船只、单独的空间站，或者是其它什么太空漂浮物，被列在了蓝色空间站列表的上方，则它们是和当前任务有关的。

Follow: Terran Scimitar

由于第一步已经跟随过了，这步直接过。

----------

Fly to: Neptuen

这时，任务刷新后出现新导航，导航标为橙色，方向标也为橙色。此时导航指向西面的 Trans-orbital Accelerator （轨道加速传送门），以后叫“星门”，如果飞船没有安装 JumpDriver，就只能从星门跨星区，很明显开始送的这艘就没有安装。手动操作飞船飞向导航目标，或者自动跟着那艘 Scimitar 也可以（慢）。

----------

Destroy: Xenon ships

跳到这里后，直接遭遇战斗。很明显，由于你的飞船根本没有安装火控软件，不可能打得有多准，不过 Xenon 船也不会攻击你，这实际上就是一次射击基础课，让玩家体会一下没有火控系统的痛苦，尽情地放空炮吧。

----------

Scan: Jump Beacon

这个时候出现一个 Jump Beacon（跳跃信标），由于这东西不会躲避，没火控系统也能摧毁它。

----------

Fly to: Oort Cloud

Wait for: Message Drone

Follow: Message Drone

流程：

摧毁 Jump Beacon 后，会接到新命令，要你去 Oort Cloud ，等待、并跟踪 Message Drone （无人间谍船，速度很快）。先不着急去，先探这里的星图，然后停靠这里的Orbital Defence station ，一来可以存盘（只能在空间站存盘），二来可以升级一下船的速度。由于刚开始只有 1000 的钱，所以要卖点东西，你那两门 Electro-Magnetic Plasma Cannon（简称 EMPC 炮）是吃饭的家伙，肯定是不能卖的，只好卖那三组 Poltergeist Missile 导弹（希望你还没用），三枚卖掉后大概能换来 10,000 左右，这个钱应该能够把引擎速度升级满了。剩下的钱应该还有3000多，就不要乱花了，要用来买 Fight Command Software MK1 的，这是火控系统，没这个东西要命的。FCS MK1 的价格是2500以内，不过现在还买不了，因为“关系”等级还不够。引擎升好后，接着做下一个任务。（可以试下存盘）

由于去 Oort Cloud 需要经过几个和此任务无关的星系，建议既然来了还是把这些星系的空间站都探了。

注：

这个时候建议先不要着急去奥特星云，先把这个星区的空间站都探测出来。因为任务是没有时间限制的，而每当你探测出一个新的建筑或者船只，除了能让他们在地图中出现方便靠港外。更重要的是它们的信息会立即进入到你的“百科全书”中，以后的好处多多。除非碰到时间有限的任务，或者是需要跟踪目标的任务以外都这样探一次。有时候时间拖太久的话，任务可能会变更，但无非是换一个星区去巡逻而已，任务内容本质上是一样的。

相关操作：

首先说下相关信息怎么看。

跟踪锁定目标时，按 F3 可以看到目标船只外形，点目标，在点 info（情报）可以看到对方详细信息。

打开点左边的 Current Ship -〉info，可以看到你当前船只详细信息，包括速度、装甲、护盾、火力、软件，等等信息都在里面。

点左边的 Personal-〉Advenced -〉Encyclopedia 就可以打开百科全书了，绝大多数信息都能找到（但不是所有的）。

交易的时候，有些货物为红色，代表你得关系不够交易这些货物，具体需要什么级别的关系，系统在交易窗口的右下角会注明。

察看自己关系等级的操作，打开点左边的 Personal -〉My Pilot Status 弹出信息，Account 你的帐户有多少钱，Titles（头衔）下能看到你的关系。至于其他统计，都是些没什么实际意义的统计而已。比如你开了几炮，命中率是多少。

再说靠港的方法，和前面任务跟随船只一样，区别在于，跟随是按 Shift + A ，而靠港按 Shift + D 。

然后说下交易操作，靠港后点左边的 Station ->Trade 就能调出交易菜单，找到 Weapons 项中的 Poltergeist Missile （窗口下方，右边蓝色代表你所在的船拥有该物品数，左边红色代表交易目标拥有该物品数），把3组 Poltergeist Missile 导弹全部卖掉（点那项货物，然后按键盘方向键，“<- 卖”“-> 买”）。然后再找到 Upgrades 项中的 Engine Tuning，全部买，就能把飞船速度升满。

存盘不用说吧？点左下角的钩-> Save，需要说的是，只能在空间站存盘。

关于这些以后就不再重复了。

Dock At: Saturn Research Station

停靠研究所，Terran 第一阶段任务就算彻底做完了。接下来可以领到一架 M5 战机，存盘吧。

#2

==========

Dock At: Orbital Defence Station in Asteroid Belt

流程：

跟着导航走，靠该靠的港（这句话以后简称“跟着导航走”），不过等少将把话说完再离港。

----------

Dock At: Shipyard in Mars

流程：

跟着导航走，该任务会得到一艘新的 M5 级别战机，建议卖掉它可以换 18,000 以上的钱。

注：

在卖 M5 战机之前，先存盘免得你卖错什么。先把它携带的装备都转到你不打算卖的飞船上，避免需要的东西随战机一起被卖掉，而且船和物资分开卖也可以赚到更多的钱。

相关操作：

在空间站转物资的方法。点左边的 Personal-〉My Property。这时弹出 ALL 页面，你所有的财产都在这里，如果需要细分，可以点其它页面。现在你可以看到，你有两艘船停在空间站里，分别是一艘 M4 级的 Sabre (M4 类似中型战机)和一艘 M5 级的 Rapier（M5 类似轻型战机）。你目前的座舰是那艘 Sabre ，左舰点 Sabre 后弹出菜单，再点 Advanced -> Freight Exchange with ...，这时出现你可以与之交换物品的船只，很明显，除了你自己这艘以外，就只有那艘 Rapier 了，点它与它交换物品。把所有东西往你的船只上转移，不过很明显，这架战机上只有1件 1MJ 护盾可以拿走。

卖船只的方法，船只必须停靠造船厂是先决条件。靠港后， Personal-〉My Property 出现财产清单后，点你想卖的拆产 -〉点 Trade 卖法和其它货物一样用键盘“<- 卖”“-> 买”。

----------

Talk To: Maj.Gen.Rai Ishiyama

跟着导航走，与 Maj.Gen.Rai Ishiyama 对话。

相关操作：

与空间站的人通话方法。靠港后点左边的 Station -> Comm -> 你想要通话的人。

----------

Talk To: Cmdr.Mark Jackson in Heretic's End

流程：

跟着导航走，不过在走之前建议先给船升下级（因为之前卖了艘 M5，现在又有钱了）。升法前面升引擎就已经说过了，就不再说了。这次升 Rudder Optimisation (提升转向能力)，如果钱够的话升满。

注：

如果有富余的钱，可以考虑买一套软件，FCS MK1 应该是你要买的第一套软件。Fight Command Software MK1 ：This is a software extension for any shipboard computer and includes a laser targeting support system.new commands are added to the shipboard  computer's command console.

翻译：这是一个软件扩充件与任何舰载电脑兼容并包含了一个支持激光锁定的系统。新的命令将附加到舰载电脑的命令控制面板中。

显然由于任务是你自己做，并不需要它“新的命令”那部分，我们主要是为了要它的“激光锁定系统”。买了之后敌对目标旁会出现一个圆形的红色准心，那个准心就是“激光锁定系统”认为你应该开火或发射导弹的方向。

如果这里没有 FCS MK1，可以去其它有的空间站买。

相关操作：

点左边的 Navigation -〉Sector Map -> 然后点你认为可能有你需要软件的空间站（通常都是在后缀为 Station 或 Dock 的空间站中）-> info，这样你就能不用靠港，就看到那里有没有你需要的东西了。

如果本星区所有的空间站都没有你需要的软件，这个时候建议先打开百科全书（如何打开前面说过了），找到 Wares 栏，点其中的 Ship Equipment，里面会列出你所发现过的所有舰载软件，如果里面没有 Fight Command Software MK1 ，则说明你还没有发现这种软件。如果里面有，则点它，进入 Fight Command Software MK1 的百科全书，找到 Available at Known Docks 项，下面列的就是已知有这种软件的地方。已知这软件的所在星系和它在该星系的哪个空间站被发现，说得很清楚。

最快捷的飞到那星系的办法：那点左边的 Navigation -〉Universe  Map -> 点你要去的那个星区 -> 再点你要去的具体空间站，弹出菜单后点 AutoPilot:Follow ，就会自动飞向那里了。这是最快捷的办法，当然你要手动飞过去也可以。

好了，关于如何不靠港的情况下，跨星区找物品买卖地，以及如何跨星区自动飞行，已经说清楚了。以后类似这种操作就不再重复了。

Waiting For: Jackson to contact you

这种任务没什么好说的，等！就一个字

Investigate: Convoy attack in Nyana's Hideout

这种侦查任务，都是跟着导航走就OK了。但不要忘了把沿途经过的空间站都探测到你的星图里，方便以后。

注：从这时起，就可能会碰到能见度很低的星区了，探测这些星区的空间站就需要点耐心了，实在没耐心最起码也要把贸易站给搜出来。

----------

Follow:  Pirate Nova Raider

流程：

这个任务要求随便跟踪一架 Pirate 的 Nova Raider，需要注意的是跟随距离，必须大于你的武器射程，但不能大于你的机载雷达搜索范围。一直跟踪到一个没有空间站的未知星区，就是海盗基地所在的星区了。这个星区不用探空间站，因为没有。继续保持距离，直到把海盗基地探出来。探出来后第一件事就是走人，等你感觉距离足够远后，再来考虑接下来的策略，免得被干掉之前努力全白费。

相关操作：

如何跟随前面说了，需要特别说的是 Shift + F ，可以使你的速度与目标的速度相匹配，但是会失去跟踪能力，所以 Shift + F 和 Shift + A 配合使用。

注：如果因为探索星图而导致目标跟丢，不用担心，过一会就会有新的同级船出现。

----------

Scan:  Pirate Base

海盗基地释放的敌船速度都无法和你引擎升满了的 M4 比，所以多绕一绕吧。目的把对方引开后，你可以靠近海盗基地至 500M 以内，进行扫描。扫描至 100% 后立即走人。

----------

Talk to Cmdr.Mark Jackson in Heretic's End

前面做过这类任务，就不再说了。

至此，第二阶段任务就算彻底做完了，得到的奖励是三架 M5 战机，存盘吧。

#3

==========

Fly To: Terran Sabre

跟着导航走

----------

Add To Wing: Your Rapiers

领你的奖励，3 架 M5。拿到战机后，让他们跟随你飞行一会，而你则跟随任务飞船飞一会，以证明你已经知道怎么指挥其它飞船，系统即判你完成了这个阶段任务。

注：

任务完成后，建议让你所有的战机都靠港，好整理一下装备。以后有了 Transporter Device 后，整理装备可以在太空中进行，但现在还没有这种装备。靠港后把尽可能多的装备转移到你的座舰(M4)上。尤其是武器 Electro-Magnetic Plasma Cannon，起初你的船上只有两门这种炮，现在每架战机上都有一门，总共就有五门了，你的座舰最多可以携带四门 EMPC，现在你可以装备满了。多余的一门既可以卖掉换回至少 300,000 的钱，也可以把它安装到其中一架 M5 上(建议还是卖了，换来的钱用来买货船，以后的主线任务会用到)没了武器的 M5，可以把它们飞到造船厂去卖掉换钱，也可以保留下来，因为他们本身价值并不高，凭你喜好。我是全卖，因为以后根本就用不到 Rapier ，如果拿来侦查的话，Rapier 的货舱是 S 型(即小型)，装不下更大的雷达，拿来作战又太不经打，每报销一架就等于损失了两门价值高达 700,000 的 EMPC，而飞机本身的价值才 20,000。另外，所有的 1MJ 护盾卖掉，你的船可以安装五块 5MJ 护盾，但刚开始只有四块，有钱后把最后一块也补上。

但这里建议卖之前可以先留一架练练编队命令。对编队操作有一定概念后再卖。卖出来的那 300,000+ 主要是用来买货船的。另外说下，各种物品需要在不同的空间站买卖，而关系的好坏直接决定你能买到什么级别的船只。考虑到关系的因素，最好是你想买哪个星区的船，就在哪个星区卖东西，这样可以赚点关系，反过来方便你买那个星区的船。建议买卖就在火星进行，因为火星船厂可以买到一艘货舱最大 4500 XL，速度 110M/S 的货船。

相关操作：

打开你的财产页面（怎么打开前面已经说过了），打开后点你的战机，这个时候最常用的命令，诸如跟随我、保护我这些命令已经在这里了，继续点 -〉Orders -〉Command Console -> 可以打开更多的命令（具体有些什么命令，取决于你给它们安装了什么软件，但最基本的命令，比如导航是一定有的），继续点 -〉Navigation -> Dock at，这时就出现了宇宙图，继续点 -> 希望去的星区 -> 希望停靠的空间站。靠港命令就完成了。依次给你的所有战机下达停靠命令，这样就可以让他们都靠在一个空间站，方便传输装备了。

但有一点需要指出的是，每个空间站的泊位都是有限的，有些多有些少。如果要求停靠的船只数量超过了空间站泊位的数量，就有可能造成“撞击事件”，严重的会撞毁你的船。所以以后有了传送装置后，还是尽可能地在太空传送货物。

----------

Fly To: xxxx

Patrol: xxxx

在继续这类任务之前，建议卖掉多余的装备和飞船，买一艘空间大于 1000，货舱级别为 XL（即超大货舱）的货船，也就是前面提到的建议在火星船厂买的那艘。再继续做这个任务，当然你不想买船要直接做也可以，只不过以后的任务还是要买。准备好后，跟着导航走。

注：

如果船厂里你想买的货船是红色，说明关系不够，在当地做点买卖、攻击这个势力的敌人、接当地的支线任务都可以增加与这个势力的关系。商船买到后，第一件事是让它停靠可以改装船只的空间站，给它安装护盾、升级引擎是必须做的两件事，忘了给它装 SETA 也会十分后悔。

当你的船队中有一艘比你操作的船只更慢的船只时（比如你让你的货船跟随你那艘 M4 时），这时你使用自动航行后，速度会自动匹配到你船队中最慢的船只，让慢船脱队即可恢复速度。

你有时候可能会碰到这种情况，在空间站等一艘速度慢的船靠港，但那船速度实在太慢等得上火，不妨先把监视窗口打开，监视该船的靠港进度，然后你退出港口，打开 SETA 让时间过快点，同时注意监视窗口，看到慢船终于要靠港了，你再进港，这样可以节约 N 倍的时间。

货船搞定后，应该还有富余的钱，可以去给自己的座舰买套新软件—— Triplex Scanner ，三倍探测范围。与此同时，把货船往 Omicron Lyrae 派，这是因为它跑得慢，让它先走，免得你老等。

相关操作：

打开你的财产 -> 点你的货船 -> 点 info ，可以看到货船的详细信息，包括它可以安装哪种级别的装甲、最多能装多少块、能装什么武器、速度能升到多少算满，等信息都在里面。也可以直接从百科全书里找到相关信息。

如果你想监视某艘不直接受你操作的船，打开你的财产 -> 点你想监视的船 -> Display on Monitor -> 出现选择，显示左边还是右边。选好后关闭窗口，就能在屏幕左/右侧看到监视窗口了（最多可同时监视两艘船），但只有你们在同一星区时才可以有监视画面，右上角的 X 可以关掉这个窗口。

相关操作：

先打开Sector Map (区域地图)，找到列表的最下面一栏，那里列出来的就是太空漂浮物。锁定后，直接用飞船去撞这些货物，货物就会自动进船舱。但要注意的是，必须弄清楚货物所需的货舱空间是否超过了你的船只船舱容量，另外还要弄清楚货物需要什么类型的货舱，XL（超大货舱）可以装所有货物，L （大型货仓）不能装 XL 货物，M (中型货舱)不能装 XL、L 两种货物，以此类推。如果船舱装不下货物还去撞，就会变成一次“真正的撞击”。如果不小心打捞到了不值钱的 Fuel，而你不想要的话，可以把它 Eject（弹射）掉，具体操作是本船为点左侧的 Current Ship -〉Freight（本船货舱）-> 找到不想要的东西点它 -> 会弹出三个选项，信息、全部弹射、只弹射部分数量。如果是弹其它船的，就从财产清单里找到相关的船，后面的操作都一样。

#6

==========

Rise To: Confirmed Friend

这是提升关系的任务，要求把关系提升至 Confirmed Friend，相信这是一个 Argon 的关系等级，因为接下来要去的地方是一个 Argon 控制的军事地区。提升关系的方法之前买船的时候就已经说过了。

注：

这实际上是一堂商人课。各种矿厂（矿场唯一需要的东西是 Energy）产主要原料、次要原料，然后把这些主、次原料卖给工厂，工厂生产出来的民用商品卖给贸易站，军用商品卖给军事基地，就是这样的。当某样商品已经稀缺时，这个时候利润时最大的时候。第一次在一个某个空间站交易某种以前没交易过的商品是一定会加关系的。所以尽可能在不同的空间站交易尽可能不同的货物是提升关系最快、最安全的办法。单就提升关系而言，交易的量并不是最重要的。

虽然我知道有更快捷的提升办法，但是过关不是目的，熟悉贸易才是这个任务的关键。

路过 Belt of Aguilar 的时候，建议去 Jonferco Headquarters 买个 Transporter Device，有了它，以后自己的船交换物品可以在太空中进行，而不再需要靠港。这里顺便还可以卖你打捞的导弹（如果你有的话）。

这个任务如果导航没有开启（仍然指向 Teladi 的空间站），说明你和 Argon 的关系还不到位。

相关操作：

如果希望以贸易来增加关系的话，建议把座舰换成商船（可以多赚点钱）。交换座舰的两种办法：1、现座舰和未来的座舰在同一空间站靠港，点希望去的船 -> Advanced -> Change Ship。2、两艘船均在太空，距离很近，这时直接点屏幕左下角的 Eject（弹射）把自己弹射出去。此时你是宇航员状态，飞行方法和飞船一样，靠近你希望去的船，点 In to the ship。如果想去的船已经安装了 Transporter Device，则五公里的距离就可以直接传送进船了。找空间站和商品的办法和之前找软件的方法一样，还找不到可以在百科全书里查该矿产地

----------

关系到位后，先派你的商船去 Teladianium 矿场买 200 Teladianium。然后把船换回来，你驾驶战机让商船跟着，因为路上可能碰到海盗。这实际上是一堂商人反海盗课，与先前不同的是这次不是保自己而是保商船。在未知星区把海盗消灭干净后再进去，免得回来的时候还要打。进 M148 后派你的商船和你的座舰，去靠那个区域唯一对你中立的空间站。进站后开始一系列的，超级漫长等待。等商船靠港后，换船进商船，这个任务即完成。
