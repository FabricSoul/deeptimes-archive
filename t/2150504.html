1.Life，生命，

以下参数控制程序生命，仅允许在Life标签中使用

Class 生命的化学类别

以下参数要输入到Class后面的""里面，注意大小写，输入""符号，需要在英文状态下，按shift+"键2次输入

示范代码 Class  "Organic"

"Organic"  有机的，以水和碳为基础的类地有机生命

"Exotic"  奇异的的生命，基于其他原则而非地球般的生命

Type 生命的发展水平:

以下参数要输入到Type后面的""里面，输入""符号，需要在英文状态下，按shift+"键2次输入

示范代码 Type  "Unicellular"

"Unicellular" 单细胞

"Multicellular" 多细胞

Biome 生物群系 ，行星上生命的栖息地：

以下参数要输入到Biome后面的""里面，注意大小写，输入""符号，需要在英文状态下，按shift+"键2次输入

示范代码 Biome  "Subglacial"

"Subglacial"  冰下，在像木卫二这样的冰天雪地的地下海洋里

"Marine"   海洋，像地球上的海洋生物

"Terrestrial" 陆地，像地球上的陆地生物

"Floaters"or“Aerial” 漂浮或者空中的，像气态巨行星的大气中的空中生命

可以通过斜线分隔多个生物群系，示范代码 Biome  "Marine/Terrestrial"

Panspermia true 泛种论是真确的，味着这个星球的生命是从另一个星球的生命形式演变而来的（例如，小行星撞击碎片

从一个行星运到另一个行星的微生物）。

可以指定两个生命标签，描述不同的生命类别。例如，一个像泰坦这样的世界，既有其地下海洋中的有机生命，也有其表

面和碳氢化合物海洋中的奇异的生命

如果没有指定生命标签，行星可以生成程序生命。如果要禁用程序生成，请在planet标签中使用此句代码：

NoLife  true

2.AccretionDisk，吸积盘

以下参数控制程序吸积盘，仅允许在AccretionDisk标签中使用

吸积盘标签，用于在黑洞，中子星或白矮星周围创建吸积盘。

nnerRadius 内半径，圆盘内边缘的半径，单位为AU。如果没有指定，则生成过程值

OuterRadius or Radius  外圆或者半径，圆盘外缘的半径，单位为AU

AccretionRate 吸积率，每年太阳质量的增加率，用于计算未指定的其他参数。

Temperature   温度，圆盘上最热部分的温度，单位是开尔文

Luminosity  光度，圆盘上以太阳光度倍数的光度

TwistMagn  扭曲强度 ,纹理上扭曲效果的强度

Density 密度或不透明度值

Brightness or Exposure 亮度或者曝光，全局亮度倍增

如果未指定AccretionDisk标签，则可以为黑洞，中子星和白矮星生成程序吸积盘。 如果要禁用程序生成，请在Planet标

签中使用此句代码：

NoAccretionDisk true

3.CometTail，彗尾

以下参数控制程序彗尾，仅允许在CometTail标签中使用

彗尾标签，用于彗星和蒸发的行星

MaxLength 最大长度，AU中最大长度的尾巴 (即彗星或行星经过它的轨道围场时)

Particles 粒子数 (精灵)

GasToDust 气体颗粒与粉尘颗粒的比率。值为0.0会生成一个完全含尘的尾部，1.0会生成一个完全气态（离子）的尾部

Bright 明亮，尾部的整体亮度

GasBright 气体颗粒的亮度

DustBright 灰尘颗粒的亮度

GasColor 气体颗粒的颜色

DustColor 灰尘颗粒的颜色

如果未指定CometTail标签，则可以生成程序彗尾。 如果要禁用程序生成，请在Planet标签中使用此句代码：

NoCometTail  true

4.Rings，行星环

以下参数控制程序行星环，仅允许在Rings标签中使用

行星环，行星环系统的标签，用于行星、卫星和小行星

纹理 - 环纹理的路径。它必须是RGBA格式并且具有x * 2像素的尺寸，其中x是纹理的长度（512,1024等）

第一行像素是前光径向颜色图案（在alpha通道中具有不透明度），第二行是背光径向颜色图案（在alpha通道中具有不透

明度）。

看看Space Engine中的土星环：当它背光时，即当太阳落后于土星时，它有不同的外观。

如果未指定Texture参数，则将生成程序星环纹理

nnerRadius 环内边缘的半径，单位为km

OuterRadius 环外边缘的半径，单位为km

FrontBright 正面亮度，当正面照明时星环的亮度

BackBright 背光，背光时光环的亮度

Density 密度，全局不透明度倍增

Brightness or Exposure 亮度或曝光，全局亮度倍增器

旋转周期 以小时为单位的圆环旋转周期

旋转偏移 以度为单位的旋转偏移（即方向调整）。

如果未指定Rings标签，则可以生成程序星环。如果要禁用程序生成，请在Planet标签中使用此句代码：

NoRings true

5.Corona，日冕

以下参数控制程序日冕，仅允许在Corona标签中使用

Period – animation period in years; the lower the value, the faster animation is.

Radius – radius of the corona in km.

Brightness – brightness of the corona.

RayDensity – density of the ray-like features.

RayCurv – curvature of the ray-like features.

恒星日冕的标签。只适用于普通恒星(即不适用于黑洞、中子星和白矮星)

Period 周期.周期动画（以年为单位）.值越低，动画速度越快

Radius 半径,日冕半径，单位：km

Brightness 亮度,日冕的亮度

RayDensity 射线密度,类射线特征的密度

RayCurv 射线密度曲率,类射线特征的曲率

如果未指定Corona标签，将生成程序日冕。 如果要禁用程序生成，请在Planet标签中使用此句代码：

NoCorona  true

6.Atmosphere，大气层

以下参数控制程序大气层，仅允许在Atmosphere标签中使用

spaceengine使用多个预先计算的大气模型，以扩展名为.atm的特殊二进制格式存储在

data/models/atmospheres/atmospheres.pak文件中。

它们在保存在同一个PAK文件中的脚本文件atmospos.cfg中进行了描述。要添加您自己的模型，您可以创建一个新的文件

addons/models/atmospheres/mymodels.cfg（文件名无关紧要），

并在其中输入大气模型脚本。请注意，并非所有计算机都能正确生成.atm文件，因为不支持几何体着色器或某些其他问题

。此外，新创建的大气模型可能只用于你的行星，它们不会出现在SE宇宙中的任何程序行星上。

因此建议只使用标准的SE模型。

标准大气模型列表：

Earth.atm  地球大气模型、正常密度、蓝天和红色日落

Mars.atm  火星大气模型、正常密度、红色天空和蓝色日落

Venus.atm  金星大气模型，厚厚的黄色天空和蓝色日落，密度很大

Jupiter.atm 木星大气模型，正常密度，白色天空和黄色日落

Neptune.atm 海王星大气模型，正常密度，深蓝色天空和粉色日落

Titan.atm 泰坦大气模型，正常密度，紫罗兰色、橙色天空和红色日落

Pluto.atm 冥王星大气模型，薄薄的，几乎是黑色的天空和淡蓝色的日落

Chlorine.atm 自定义的大气模型，厚实、绿色天空和绿色日落

Thick.atm  自定义的大气模式，厚厚的白色天空和橙色日落

Sun.atm – 自定义大气模型，薄型、白色天空和白色日落（(产生的颜色是跟恒星的光的颜色匹配）

Model 大气模型，在这个参数输入面列出的大气模型的名字，或者创建的大气模型名字

输入到Model后面的""里面，输入""符号，需要在英文状态下，按shift+"键2次输入

示范代码 Model"Earth"

Height 高度，大气顶界的高度(以公里为单位)，类地行星的典型值为50-100公里，气态巨行星为500-2000公里

Pressure 气压

Density  密度，零高度处的空气密度，单位为kg/m3

MolarMass 摩尔质量，空气的平均摩尔质量，g/mol

Adiabat 绝热，绝热指数

Greenhouse 温室效应温度，单位开尔文

Bright 亮度，默认值是10

Opacity 不透明度，渲染不透明度。要避免视觉假象，请使用默认值1

SkyLight 天空光，白天天空照亮行星地表的亮度。默认值为3.3333

Hue 色相，改变颜色的色相。默认值是0

Saturation 饱和度，

Composition 颜色饱和度的变化。默认值为1

Composition 描述大气化学成分的标签。标签必须包含以气体命名的几个数值参数；值是这些气体的百分比

地球的大气成分示范代码

Composition

{

N2  77.7729

O2  20.8625

Ar  0.9303

H2O 0.4000

CO2 0.0398

}

支持气体有：H2、He、Ne、Ar、Kr、Xe、O2、N2、Co、CO2、SO、SO2、Cl2、NaCl、H2S、H2O、NH3、

CH4、C2H2、C2H4、C2H6、C3H8。SpaceEngine可以根据大气成分计算密度、摩尔质量、辐射和温室效应温度

如果未指定大气标签，则可以生成程序大气。 如果要禁用程序生成，请在Planet标签中使用此句代码：

NoAtmosphere  true
