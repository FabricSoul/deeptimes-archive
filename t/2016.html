Launcher设定介绍！看完下面的再玩！

主要说说Features页里面的几个设定，重要点的就说，杂项太多就不说了，我英文太烂也说不清:lol 。

Launcher.exe在Freespace2的游戏目录下能找到！

运行Launcher.exe后，先点右上角“Browse”按钮，确定“fs2_open_3_6_9.exe”这个文件的位置，然后点右下角“Apply”按钮。

接着点Features那一标签页。

在Easy右边的下拉菜单选择“All features on”，一般主流机子的性能选All features on是没问题的。机子不太行的话可以选High memory usage features off。

List右边的下拉菜单：

1.Graphics，跟画面有关的选项。不熟悉的话照上面说的选All Features on就行了。

2.Game Speed，选上的话能提高游戏速度.

3.HUD，座舱界面选项，都选上吧，多了原版没有几个东西，如3的雷达，扫描线，计时器等等。

4.Gameplay，建议都选上。

5.Audio，建议选上第一个Preload mission game sounds，能减少游戏中偶尔突然卡一下的情况。

6.Multi，联机选项，还没能找到人跟我联机，没碰过:Q

7.Troubleshoot，游戏出错时才能用到的，很多选项，头晕，偷懒不说。不知道原因的话根本不会用。

8.Experimental，不懂不说。

9.Dev Tool，有两个有用的选项。

---Show Frams per second on HUD，在座舱显示帧数。

---Run in window，窗口模式我喜欢。

MOD的选择：

-----1、最好每个MOD对应一个人名（就是开始进入游戏时你能创建的），减少出错的机会。

-----2、玩原版任务的话，在MOD那个标签页点SelectMOD，然后选MEDIAVPS，画面会漂亮很多哦。

-----3、选择其他MOD的话，进入游戏后，先进入Campagin Room，然后选择你所选的那个MOD的名字，然后点右下角的Select。

-----4、打完原版任务后建议先玩INFERNO R1 MOD。运行Launcher.exe，在MOD那个标签页点SelectMOD，然后选INFR1。进入游戏先点Campaign，选INFERNO:GTVA CHAPTER 1 然后点SELECT出去。点CONTINUE开始玩了！

TECE那项进入可以看到所有的船只、战机、技术、你通过的任务等等，在任务那页你可以选择并重玩你曾经通过的所有任务。INFERNO R1 MOD 里面的战舰飞机比较帅啊，好好欣赏一下！

1.用daemon载入，自动播放

2.安装Freespace SCP，里面还有附带几个MOD，目录最好是 X:\Games\Freespace2 （X：是指你要安装的硬盘分区的盘符）。运行X:\Games\Freespace2\Launcher.exe，在选项卡的顶部有个Browser按钮，点击然后选定X:\Games\Freespace2\fs2_open_3_6_9.exe这个文件。之后就是设定Video，Mod，等等。这个游戏配置要求很低，主流机子选最高都绝对没问题。

（可选）3.安装Inferno，要安装在X:\Games\Freespace2目录下，然后运行X:\Games\Freespace2\Launcher.exe，然后在Mod里点Select Mod按钮，选择X:\Games\Freespace2\INFA目录，这样就选定了这个MOD了。我试了下这个MOD好像有点问题，不知道是不是我装了两个Freespace2的问题。如果不想碰运气的话就别安装，请直接复制BT下载下来的INFA目录到X:\Games\Freespace2 目录下。

（可选）4.The Babylon Project 这个是独立的，不安装1.2,3步也行。看过美剧巴比伦5号的就应该来玩一玩了，嘿嘿。

（可选）5.Wing Commander Saga Prologue 这个也是独立的。

安装完成后进入游戏所在的目录，运行里面的oalinst这个文件。如果没有这个文件就去http://developer.creative.com/ar ... p;top=38&aid=46下载oalinst.exe这个文件，然后安装它！
