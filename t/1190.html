说明:

1.EngineTuning是EqD升级速度的部件，其实大家可以看到，它代表了最大速度和最小速度的比值，例如10的EngineTuning，就是可以提高100%速度，5的EngineTuning就是可以提高50%的速度，以此类推。

2.武器能量恢复率只要乘于武器能量，就可以得到每秒武器能量恢复的Kw数了，表格实在放不下，需要的人可以直接在Excel中很方便的用乘法添加。

3.Rudder是飞船灵活度的升级指标，这里提供了可升级数量的数据，可以和更详细的”偏航“”转向“”翻滚“数据一起作为考察性能的指标。

-----------------------------------------------------------------------Hammer的分页线------------------------------------------------------------------------------------ 表格华丽的登场了，表中的数据是我直接调用2.0.02版本中的cat文件，逐个获得，由于量大。到后来眼都花了，如果有发现错误请告诉我，谢谢！

Okay，先介绍一下表格中数据的含义

A. 种族和船名：这个不要多说什么了，X3人都知道。

B. 下面说下级别和分型的含义

TP : 大型客机

TS : 运输机(这个就是我们发财致富的基础，货船啦)

TL : 空间站运输船

M1 : 航空母舰

M2 : 巡洋舰

M3 : 重型战斗机

M3+ : 改进型重型战斗机(完成BalaGi任务后才能获得，表格中用红色字体标出的)

M4 : 轻型战斗机

M5 : 侦察机

M6 : 护卫舰

M7 : 驱逐舰(完成BalaGi任务后才能获得，表格中用红色字体标出的)

同一种型号的飞船可能有不同的版本，介绍一下常见的

Raider：歼击机

Vanguard：先遣机

Sentinel：护卫机

C. 速度单位是m/s，加速度单位是m/s²，这2个也不用多说了哇

D. 灵活度，分偏航/转向/翻滚，3项指标，数字越高越灵活啦，单位是rot/min。

E. 最大货舱，就是能装的货物量。货舱级别，因为物品大小分S/M/L/XL/ST(S最小，ST最大，和衣服尺寸一个概念)，所以要相应的货舱来容纳，当然向下兼容的啦。

F. 最大护盾，就是该船能装的最大等级的护盾(护盾从小到大有1Mj/5Mj/25Mj/125Mj/1Gj/10Gj)以及能安装的最大数量。这里要说明一下，护盾能量和护盾本身，决定了护盾恢复速度。相同护盾情况下，能量越高，恢复的也越快，但达到最大恢复速度后就不再提高恢复速度了。实际使用中主要就M1/M2我们可能关心这个速度，所以恢复时间没有列在表中。关于恢复时间我还不知道具体怎么计算的。列一些数据供大家体会一下：（护盾恢复时间现已加入表格数据中，２００７/０１/２７）

护盾能量：100kw，25MJ*3,恢复需要834秒

护盾能量：250kw，25MJ*3,恢复需要336秒

护盾能量：300kw，25MJ*3,恢复需要295秒

护盾能量：627kw，25MJ*3,恢复需要167秒

护盾能量：715kw，25MJ*3,恢复需要128秒

护盾能量：750kw，25MJ*3,恢复需要112秒

护盾能量：780kw，25MJ*3,恢复需要112秒

护盾能量：1125kw，25MJ*3,恢复需要112秒

护盾能量的单位是kw。

G. 武器能量，武器除了MassDrive外，都要消耗能量，每种武器消耗的程度不同，但飞船能提供的能量更关键，这是反映飞船持续战斗力的一个重要指标。单位是MJ。
