我们都喜欢MARS火控系统，我们都喜欢中文脚本，我们都喜欢稍微比原版加强点的DD（是你自己喜欢修改吧- -!）

看到好多人对英文版的MARS不明白如何设置，还在使用老版本的汉化MARS。

为了方便大家，方便自己，所以这个中文威力加强版MARS4.95出炉了。（和日本光荣学的，PK中文版就骗钱的DD）

脚本名称：MARS火控系统汉化加强版

版本：MARS/CMOD4/Engine Colors/Other scripts

作者：GAZZ（MARS）、fud（Engine Colors）、paulwheeler（CMOD4）etc...

汉化版本制作：玺睿

X3适用版本：X3TC2.71

EGO官方论坛链接：http://forum.egosoft.com/viewtopic.php?t=216690

汉化加强版特别说明（必看）：

1.MARS4.95完整汉化，修正上一个版本的排版问题，对汉化文本重新润色。

2.整合了CMOD4武器平衡MOD，MOD对X3TC中大部分武器的效果作了修改。对武器参数做了平衡性处理。（包括护盾，炮塔文件）

3.整合了Engine_Effects（飞船尾焰效果），与CMOD4的材质和武器效果兼容。（只适用于此版本，不兼容安装修改TSHIPS文件的MOD的X3TC）

4.对AI控制的船只（非自己财产飞船）的实弹武器的弹药供给做了脚本。（因为电脑的供给系统不健全，AI战舰货舱经常性的缺少可用弹药导致战斗能力莫名削弱，此脚本会为缺少可用弹药的电脑飞船添加弹药）

5.移除默认飞船名称“你的”。这个可以在游戏设置中的人工智能选项中选择开启或关闭。

6对X3TC的OOS战斗判定的算法做了平衡性修改，OOS战斗武器的DPS不完全等于DPH。(使用GAZZ的脚本，不要担心平衡性问题。使用MARS的飞船OOS战斗会获得修正加强，原版OOS判定的问题见此贴http://bbs.deeptimes.org/forum.php?mod=viewthread&tid=38001&extra=page%3D1)

7.舰船尺寸说明：M5 = 1 :: M4 = 4 :: M3 = 8 :: TM = 8 :: M8 = 10 :: M6 = 14 :: M7 = 35 :: M2 = 55 :: M1 = 60 :: M0 = 80.

8.兼容性说明：此整合版MOD对新飞船或新武器MOD、其他飞船效果MOD、X3TC系统平衡性、命令控制台修改MOD并不兼容。对硬件优化、X3TC Bonus Package 官方脚本礼包、comp_cleaner类MOD、类似CommunityPlugin和ImprovedBoarding的AI行为改进、IR和pirate.guild3全局性参数修正增强等类型MOD完全兼容。

注：脚本代码参考和使用了许多EGO官方论坛会员的成果。向这些大神致以敬意~

效果说明：

受X3TC的引擎所限制，飞船“尾焰”效果并不算完美，所以大型船只没有添加尾焰。不过战斗画面个人感觉依然比以前要好许多。比如导弹飞行会带有长长的轨迹，就像超时空要塞中的导弹一样。看着战机和导弹带着飞行轨迹互相追逐，挺有趣的。CMOD4对武器特效的修改也不错~比如战舰用大型激光加农炮，不再像以前全是不间断的脉冲射线，有点高达SEED中大天使号的阳极粒子加农炮的感觉。

尾焰的图要包括整个飞船，图很大，所以用相册外链

安装方法：

复制所有文件到X3TC安装目录，.dat和.cat文件的序号修改为最后一位，再打汉化补丁。

即：MARS的dat文件要放在最后面，但要在汉化dat之前。例如汉化dat最后为20.dat，MARS的.dat文件改成20.dat，汉化文件的dat改成21.dat

Troubleshooting ：

If you have downloaded a script/mod and it seems not to work make sure you have followed the following 8 steps to happiness :-

1.Have you enabled the script editor by typing "Thereshallbewings" as your Pilot name? (Read The Activating The X3 Script Editor Stickie for more help)

2.Have you read the guides that are stickied on this forum

3.If the script is an AL Plugin have you saved your game after installing it and then reloaded to check if the AL Plugin settings are showing up?

如果你下载一个脚本或MOD而它们却没有正常工作，你必须确定你已经执行了以下8个步骤

1.第一次进入游戏修改玩家飞行员姓名为“thereshallbewings”并启动脚本管理器？

2.你是否认真的阅读论坛脚本的使用指南或注意事项？

3.如果这个脚本是AL 脚本（人工生命脚本，包括MARS，IR，RRF都属于AL脚本），你在第一次保存游戏进度后再读取进度，再检测是否看见了AL 脚本设置菜单？

刚进入游戏OOS combat之所以没有检测到MARS，那是因为你没有修改飞行员姓名，启动脚本管理器。当然，你也可以略过此步骤。进入一所空间站保存游戏，然后读取刚才保存的进度，你会发现读取之后AL 脚本就能启动（包括MARS，IR），设置好这些脚本之后再保存。第二次读取进步，一切脚本就可以完全正常工作（OOS combat正常检测到MARS）。

这样可以提高MOD的兼容性，避免出错的可能。

注：目前有一个问题是X3Editer2并不是万能的，对某些数据无法进行有效修改。

例如下图红线部分圆圈标注的数据-无人机MK2默认携带安装的武器数量

(并不一定等于最大武器数目)

这个只能使用EGO的官方MOD制作软件将TShips文件解压出来修改并替换掉

但可以用X3EDITER2增加炮台数目简单的达到同样效果

例如2*3个武器位默认会安装4门PAC，3+2个武器位会默认安装3门PAC

4.95原版的MARS_Repair_Drones默认是1*2个武器位，携带2门PAC

这点只是说明一下，不满意我修改的无人机MK2数据可以自行进行修改

由于“哥布林”是一种在西方神话故事里的生物，在东方语言并没有等同的概念。

特点是个性贪婪又卑劣，性格倾向邪恶、狡猾而善于诈欺。

战斗能力弱小但会以大批群体行动补其不足。
