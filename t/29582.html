1.        基本操作和上手指导。

（1）前序

最近发现x3家族里面出现了很多新人，我是十分的开心，毕竟家族又壮大了，但是呢x3的上手难问题缺难道了，所以我想老鸟都是从菜鸟来的，所以写了这样一个新手指导，望老鸟们能指出我的缺点以便不误导新人。废话不多说了进入正题。

当你进入x3的世界首先你面临的选择就是选择人物，是自定义呢，还是easy，这里可以告诉大家，建议新手先玩起手地球人的人物原因很简单，因为上手容易，因为以前就发生过很多新人起手去打海盗。。然后惨不忍睹。为什么呢？因为新人不会上手该船的性能和买炮，然后出去就被海盗虐，久而久之就不想玩了。而地球人起手的人物船的性能非常出色在先期来说是非常不错的了。

好选定人物以后进入游戏，现在开始了解基本按键，玩家最常用的就是：

TAB 直接加速到最高速度（性能决定能加速到多少）

BACKSPACE  急刹车（注意惯性，太近还是会撞。。。。）

Z  手动减速

X 手动加速

F1 切换目标（等你开上大船以后F1你会经常用因为他能切换船上的炮台）

F2 观看自己船的正面和屁屁

F3 切换到你选定目标

空格 这个按键我给大家详细讲下他的用途，空格键使用一次以后，会切换到镜头跟随鼠标模式，也就是传说中的第一人称射击模式，这个模式新手们比较喜欢用于空中搏斗吧。  空格键的另外一种模式就是按键镜头跟随模式，也就是说，鼠标可以随便晃镜头不动，为什么要这个样的模式呢，因为这样的话可以方便大家飞行中接任务等等等等。

鼠标左键 ：点击用

鼠标右键： 开火

, ：打开星区总图

。：打开你所在的星区地图

M：切换/装卸导弹

L：发射导弹

J：传说中的光速移动（简单来说就是帮助玩家节约时间，加快游戏速度默认是6倍，玩家可以在设置中换到10倍）

T-自动搜寻目标

Shift+A自动驾驶

Shift+D自动停靠

Shift+J跳跃

按键就说到这里了，如果有什么不知道的话可以问我或者各位老鸟们。

（2）导弹和枪炮

知道了基本操作有些人就想体验一下太空战斗的感觉?  先别急，基本用上手的船不经过调整去打海盗多数新手会失望的结束游戏，大家拿到上手的船了，然后可以点击下位于屏幕下偏左的玩家船只信息，为什么要看呢，因为要了解这船能装什么家伙，买错了家伙可是很亏的，然后也许很多人会问，我看中了xxxxxx武器，哪买呢?你不是说能改装性能哪改呢。大家可以使用句号键，打开自己所在地图，看到一个叫改装船坞的地方了吗?别犹豫了进去吧。一共能改三项，1最大速度，2船只灵活度，3最大货仓。至于枪炮的资料我给出当年有垂大给出的资料给大家参考http://bbs.deeptimes.org/v ... %26amp%3Btypeid%3D1买完武器大家要装武器了吧，单机自己船，选项里有个叫高级的选项，然后武器，选择，看到了没，选择炮位，装上买来的武器ok出去干架吧。

至于导弹，我相信新手们不太会使用导弹去攻击敌人吧，为什么?很简单，因为每颗都是钱=。=  但每个新人都会慢慢自己摸索所以我也特别列出链接给大家参考http://bbs.deeptimes.org/v ... %26amp%3Btypeid%3D1还有哦，导弹的产量很少，特别是威力大的导弹，如果新手们也希望自己能开上m7m体验上百导弹几十秒一起发出的感觉。自己开厂啦(以后会说)

（3）基本逃生以及护盾

经过了上面的指导大家是不是发现一般跟你同级别的船打起来十分轻松了呢，但是打高1级的船会发现吃力很多呢?实际上，各位是可以的，(看到明显大你许多的船就别想了=。= 骨灰除外)。只是欠一点东风而已。

按键Q,W,E,A,S,D都是闪避键，怎么用呢。我打个比方，Q像左翻滚W向上平移D像右平移，3键一起按住，大家会发现自己在空中划出很美丽的圆圈，但是小型船体积比较小，闪躲动作可以靠速度弥补掉，这些，只能靠大家不断战斗的经验才能得出有效的适合你的闪躲动作。

告诉新手一些基础知识，船体损坏一定要修船打架的时候船体耐久的重要程度大于护盾耐久，为什么，因为船体受损影响速度一被人打到没速度的话，要么跳跃，要么等game over=.=

跳跃：如何跳跃呢？想要跳跃的话，先要有声望够买跳跃软件和跳跃引擎，买完这些才能跳跃，跳跃时消耗能量电池，跳1星区=10能量电池。跳跃距离越大，电池的消耗就越高。

护盾呢，也非常的简单，但是呢大家不能带错盾为什么呢，打比方现在论坛中各位大大很喜欢使用的M7老虎，它的最大护盾是4G有些新手会去买2个2G护盾，然后说。。。为什么，所以请大家买盾前看清楚4*1G还是5*2G。

有些新手也许会问到，盾怎么安装在飞船呢？ 没关系你买了盾以后放在船舱中就算自动装备了哦。

(4)赚钱门路与任务

大家体验完了x3刺激的空战应该想说了吧，看xxx星球里面飘的船都比我的牛为什么我就是小船，我要赚钱买大船啊，现在我就来为大家讲下赚钱之道，总结出来3种初期赚钱的办法，2种后期赚钱的办法，今天先为大家讲3种初期赚钱的办法。其中2种是参考各位骨灰的玩法，还有一种虽然非常可怜但也不所谓一种简单的方法。

1.        战斗拣船和任务（包括任务战斗）：大家可能都看了帖子,X3的任务现在已经非常出色了，除了主线任务（通常用蓝色标志表示）还有各个空间站的护送，暗杀，保护，巡逻，计程车，送货，跟踪，拣船，造场等等等等任务，奖励也是非常丰厚的。任务难度在交谈面板中都有显示，建议新手开始在星区防守比较好的地方转转做些保护任务，为什么呢，大家试想一下，你在一个无人防守区域做保护，突然发现这些敌人不是你一个能对付的。。然后又舍不得那钱。。拼了吧。。最后读档。然而在防守比较好的地方就不会这样了，因为有国家军队嘛，打不过我们找帮手一起扁啊。久而久之大家可以从开始的m4-m3-m6，然后困难的任务中磨练自己吧。钱不用担心，绝对多。

2.        第一中方法已经给出，但是有些玩家说，我看大大们的帖子，万一我打到了自己人还要和他们通话道歉，我就是看不惯他们以后我有钱了买舰队过来灭灭灭他们。小的我送上第二种办法，做侠士。。。。玩家可以到某些海盗十分猖狂的星区去灭海盗，有什么好处呢?好处大大滴好，第一打海盗官方会感谢你（声望）但是你在什么族的地方打，只有这个族会感谢你，第二，海盗小队比较小，可以打的他们放弃自己的船，然后当然我们去接管咯，这边又涉及到另外一个问题，如何接管船只，首先告诉大家m5,m4,m3这类的小船是会放弃的，大的船。所以不要看国军在打m7大家想：哇塞大船抢了。在这边我友情提示新手玩家，生命诚可贵。废话不多了，接管首先大家把船开到离开无人船近点的地方，别撞了心疼死你。然后下船，恩？怎么下船？告诉大家在左面的快捷里面有。然后宇航员慢慢飞过去。。。别飞太快。。宇航服的刹车可没你座驾的那么好=。=撞了你肯定死。离船很近了吗?点击船，看到接管了吗，点它，ok它归你了。那大家会说，m7,m2,m1这种大船我们不能抢吗，太不好玩了，告诉大家能抢，但是需要的配备和前期工作很多，运气也是主要原因，所以，不久的将来我会给大家专门介绍抢大船的方法。

(4)各类舰船，级别。

各位玩家学会了操作，赚钱，手头上也应该有点小钱了，那么有了钱以后会做什么呢?不用说也明白（自己开的船太老土了，想换个大的）。好的接下来呢我给大家讲下舰船的级别。

TP : 大型客机

TS : 运输机(货船)

TL : 空间站运输船

M1 : 航空母舰

M2 : 巡洋舰

M3 : 重型战斗机

M4 : 轻型战斗机

M5 : 侦察机

M6 : 护卫舰

M7 : 驱逐舰

M7M:导弹驱逐舰（顾名思义，导弹解决一切）

M8:导弹护卫舰

然后大家会说，我手头这点钱正好能买m7呢，我去买吧，这边我友情提示大家其实，先买齐大船用的零件比先买大船重要，为什么呢，造船场可没那么好心给你造完船以后送你盾送你炮=。=，相反什么都没有，如果钱只够刚刚好而且没买零件的话呢。。。你的m7只是空壳而已。只有装满盾，炮的大船才能成为真正的宇宙霸主！

另外告诉大家，从m3开始某些船已经开始出现炮台，何谓炮台，就是除了你可以控制的主炮以外还有其他炮位，如果大家认为自己可以控制的过来的话就不用看我后面那段话了，如果不行的话我们可以让电脑帮我们开炮，怎么让炮台自己打呢？点击自己船的图标（中间往下）命令，看到炮塔设置了吗，里面有不同的选择，比如保护船只，攻击所有敌人，攻击玩家目标，等等，怎么设置呢？要看不同炮位的炮来决定设置，FAA这样的对空炮玩家完全可以设置攻击所有敌人而如果玩家想让所有的大型炮帮你一起集火目标击沉对方大型舰船呢，就可以设定比如IBL,PSP,IC这类炮攻击玩家目标。怎么样设置的好，设置的合理要靠玩家自己摸索，我只是给大家一个建议而已。

这样说好了，也许有些人会问，m7和m2有什么区别呢，2个都是大船呀，真难选择，好，这样说好了，m7有着自己的优势和弱势，优点大船中间m7跑的快，火力猛，转向好。弱点呢，很明显，盾少，能量不够，面对大量飞机的时候不得不选择除了主炮以外都使用防空。

m2呢，当然盾厚，能量不用担心不够，弱点呢，跑的慢，玩家不能直接操纵主炮，转向慢等等。

那有些玩家会说那m1好了，自己能打还有“警卫队”（指小飞机）。对m1带满飞机的确很猛，面对对方的任何目标可以不用畏惧，但是。。m1的造价不凡（加上小飞机的造价）外加维护费用惊人使得很多人都放弃了使用它。

关于M7M的问题，我在这边可以很负责的告诉大家。如果你们有的是钱，能开自己的工厂群自己造导弹自己用，一个M7M足够扫平一个星区。有兴趣的玩家可以去尝试下。

舰队：如何组建自己的舰队如何快速操作舰队，首先大家如果要组建舰队最重要的就是有足够的经济实力，有钱才能打仗嘛，再来就是合理的搭配，比如2艘m2 4艘m7组成的扫星区舰队，或者m1+m3+m2+m7的混合舰队。

如何快速指挥舰队：x3tc提供了强大的编组功能,你可以将你的船分组，如何分组？(舰船餐单里面找找啦，然后比如分个白组，红组什么的）就跟war3 差不多的编队系统，可以像所在编队内船只群发命令。当然也可以通过一些脚本来实现。

好了船都介绍完了，下次给大家介绍工厂与工厂群。

（1）前序

（2）导弹和枪炮

知道了基本操作有些人就想体验一下太空战斗的感觉?  先别急，基本用上手的船不经过调整去打海盗多数新手会失望的结束游戏，大家拿到上手的船了，然后可以点击下位于屏幕下偏左的玩家船只信息，为什么要看呢，因为要了解这船能装什么家伙，买错了家伙可是很亏的，然后也许很多人会问，我看中了xxxxxx武器，哪买呢?你不是说能改装性能哪改呢。大家可以使用句号键，打开自己所在地图，看到一个叫改装船坞的地方了吗?别犹豫了进去吧。一共能改三项，1最大速度，2船只灵活度，3最大货仓。至于枪炮的资料我给出当年有垂大给出的资料给大家参考http://bbs.deeptimes.org/v ... %26amp%3Btypeid%3D1买完武器大家要装武器了吧，单机自己船，选项里有个叫高级的选项，然后武器，选择，看到了没，选择炮位，装上买来的武器ok出去干架吧。

至于导弹，我相信新手们不太会使用导弹去攻击敌人吧，为什么?很简单，因为每颗都是钱=。=  但每个新人都会慢慢自己摸索所以我也特别列出链接给大家参考http://bbs.deeptimes.org/v ... %26amp%3Btypeid%3D1还有哦，导弹的产量很少，特别是威力大的导弹，如果新手们也希望自己能开上m7m体验上百导弹几十秒一起发出的感觉。自己开厂啦(以后会说)

（3）基本逃生以及护盾

经过了上面的指导大家是不是发现一般跟你同级别的船打起来十分轻松了呢，但是打高1级的船会发现吃力很多呢?实际上，各位是可以的，(看到明显大你许多的船就别想了=。= 骨灰除外)。只是欠一点东风而已。

按键Q,W,E,A,S,D都是闪避键，怎么用呢。我打个比方，Q像左翻滚W向上平移D像右平移，3键一起按住，大家会发现自己在空中划出很美丽的圆圈，但是小型船体积比较小，闪躲动作可以靠速度弥补掉，这些，只能靠大家不断战斗的经验才能得出有效的适合你的闪躲动作。

告诉新手一些基础知识，船体损坏一定要修船打架的时候船体耐久的重要程度大于护盾耐久，为什么，因为船体受损影响速度一被人打到没速度的话，要么跳跃，要么等game over=.=

跳跃：如何跳跃呢？想要跳跃的话，先要有声望够买跳跃软件和跳跃引擎，买完这些才能跳跃，跳跃时消耗能量电池，跳1星区=10能量电池。跳跃距离越大，电池的消耗就越高。

护盾呢，也非常的简单，但是呢大家不能带错盾为什么呢，打比方现在论坛中各位大大很喜欢使用的M7老虎，它的最大护盾是4G有些新手会去买2个2G护盾，然后说。。。为什么，所以请大家买盾前看清楚4*1G还是5*2G。

有些新手也许会问到，盾怎么安装在飞船呢？ 没关系你买了盾以后放在船舱中就算自动装备了哦。

(4)各类舰船，级别。

各位玩家学会了操作，赚钱，手头上也应该有点小钱了，那么有了钱以后会做什么呢?不用说也明白（自己开的船太老土了，想换个大的）。好的接下来呢我给大家讲下舰船的级别。

TP : 大型客机

TS : 运输机(货船)

TL : 空间站运输船

M1 : 航空母舰

M2 : 巡洋舰

M3 : 重型战斗机

M4 : 轻型战斗机

M5 : 侦察机

M6 : 护卫舰

M7 : 驱逐舰

M7M:导弹驱逐舰（顾名思义，导弹解决一切）

M8:导弹护卫舰

然后大家会说，我手头这点钱正好能买m7呢，我去买吧，这边我友情提示大家其实，先买齐大船用的零件比先买大船重要，为什么呢，造船场可没那么好心给你造完船以后送你盾送你炮=。=，相反什么都没有，如果钱只够刚刚好而且没买零件的话呢。。。你的m7只是空壳而已。只有装满盾，炮的大船才能成为真正的宇宙霸主！

另外告诉大家，从m3开始某些船已经开始出现炮台，何谓炮台，就是除了你可以控制的主炮以外还有其他炮位，如果大家认为自己可以控制的过来的话就不用看我后面那段话了，如果不行的话我们可以让电脑帮我们开炮，怎么让炮台自己打呢？点击自己船的图标（中间往下）命令，看到炮塔设置了吗，里面有不同的选择，比如保护船只，攻击所有敌人，攻击玩家目标，等等，怎么设置呢？要看不同炮位的炮来决定设置，FAA这样的对空炮玩家完全可以设置攻击所有敌人而如果玩家想让所有的大型炮帮你一起集火目标击沉对方大型舰船呢，就可以设定比如IBL,PSP,IC这类炮攻击玩家目标。怎么样设置的好，设置的合理要靠玩家自己摸索，我只是给大家一个建议而已。

这样说好了，也许有些人会问，m7和m2有什么区别呢，2个都是大船呀，真难选择，好，这样说好了，m7有着自己的优势和弱势，优点大船中间m7跑的快，火力猛，转向好。弱点呢，很明显，盾少，能量不够，面对大量飞机的时候不得不选择除了主炮以外都使用防空。

m2呢，当然盾厚，能量不用担心不够，弱点呢，跑的慢，玩家不能直接操纵主炮，转向慢等等。

那有些玩家会说那m1好了，自己能打还有“警卫队”（指小飞机）。对m1带满飞机的确很猛，面对对方的任何目标可以不用畏惧，但是。。m1的造价不凡（加上小飞机的造价）外加维护费用惊人使得很多人都放弃了使用它。

关于M7M的问题，我在这边可以很负责的告诉大家。如果你们有的是钱，能开自己的工厂群自己造导弹自己用，一个M7M足够扫平一个星区。有兴趣的玩家可以去尝试下。

舰队：如何组建自己的舰队如何快速操作舰队，首先大家如果要组建舰队最重要的就是有足够的经济实力，有钱才能打仗嘛，再来就是合理的搭配，比如2艘m2 4艘m7组成的扫星区舰队，或者m1+m3+m2+m7的混合舰队。

如何快速指挥舰队：x3tc提供了强大的编组功能,你可以将你的船分组，如何分组？(舰船餐单里面找找啦，然后比如分个白组，红组什么的）就跟war3 差不多的编队系统，可以像所在编队内船只群发命令。当然也可以通过一些脚本来实现。

好了船都介绍完了，下次给大家介绍工厂与工厂群。
