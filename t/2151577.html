宇宙模拟器Space Engine 0.990教程2：基本设置，浏览观光，天体基本操作

提示：

Space Engine里面的太阳系行星和卫星都使用的真实实拍纹理，所以清晰度远不如太阳系以外的程序行星和卫星

点击Planetarium（天文馆），再点击Continue（继续）进入Space Engine 0.990主界面

开始设置， 主要设置以下参数，根据你自己的电脑配置和需要设置。

2.1 基本设置

点space engine主界面左边菜单栏里的第12个图标（齿轮图标）进入Settings(设置)对话框

2.11 点击General（常规设置），点击Screenshots format（截图输出格式）右边的长方框选择png,这种格式的截图质量比jpg高

选中Auto horizon on planets（着陆时自动对准行星地平线），当登陆行星时，系统会自动调整相机处于水平，关闭这个选项，相机处于自由模式

2.12 点击Camera（相机），点击Photo mode（照片模式 ）分别有这3种选项：

HDR 高动态范围图像

Manual 手动

Auto 自动

当系统默认的曝光太亮或者上面3种曝光方式太亮的时候，适当把Photo mode（照片模式 ）下面的Exposure compensation（曝光补偿）滑条向左拖动，可以降低曝光

在Exposure compensation（曝光补偿）下面的几个参数控制行星，恒星，星团，星云，星团，深空天体的极限星等，如果想让哪种类型的天体在界面内显示的更明显，就把那个类型天体的滑块向右拖动。如果想让哪种类型的天体在界面里面不明显，就把那个类型天体的滑块向左拖动。

点击Reset to default（重置为默认值），可以恢复默认的数值

继续往下看会看到Comet tails（彗尾），Aurora（极光），拖动它们各自的滑块，可以控制各自的亮度。

点击Reset to default（重置为默认值），可以恢复默认的数值.

2.13 点击Graphics（图像），点击Graphics quality（图像质量）会有3个选项，分别是：Low 低，Medium 中等，High 高，Ultra 超高，Custom 自定义

如果你的电脑配置差，选择Low 低

如果你的电脑配置可以，选择中等

如果你的电脑一般，选择Custom 自定义，当选择这个的时候，下面的相关参数可以选择，根据主要设定各参数

如果你的电脑配置非常好，选择High 高

Ultra 超高，适合电脑配置超高的的

把Landscape LOD（地形细节层次）的滑块滑到最右边为1，地形会有效增加，提高地形质量。

当登陆行星地表的时候，右上角会显示一个转动的圆圈，当圆圈转完消失后，当前显示的地形就会加载完。

Landscape loading speed（地表贴图加载速度）一般使用默认的数值，如果你的电脑配置足够好，可以适当增加数值。

当Graphics quality（图像质量）设置为Custom 自定义的时候，会显示相关设置参数。如果电脑配置不够好，可以选择Compress textures（压缩纹理）

2.14 其他参数根据需要自己可以设置，设置完后，关闭Settings(设置)对话框

在space engine主界面右边一排的图标分别是控制飞船，星系，行星，恒星，星云，行星，彗尾，大气，云，水，极光等的设置，点击相关的图标会开启和关闭它们的显示。

2.2  浏览观光

2.21 在Space Engine中的浏览观光基本操作

按W键，是前进

按S键，是后退

按A键，是左移

按D键，是右移

按R键，是上移

按F键，是下移

按住鼠标左键，可以上下，左右拖动

鼠标滑轮向前滑动是加速，操作的时候在界面上会显示到速度

鼠标滑轮向后滑动是键速，，操作的时候在界面上会显示到速度

可以左手按住键盘上的W，S，A，D键，同时右手按住鼠标滑轮加速或者减速，这样可以在Space Engine中的宇宙中飞行

点击space engine主界面右边第17个图标，一个圆圈中间有原点的图标，可以开启速度方向（速度矢量），在天体附近操作的时候，会显示速度和当前天体坐标。

更多基本操作，请参考这个

https://baike.baidu.com/item/Space%20Engine/12009594?fr=aladdin

改变Fov（视场角）

按Page UP键可以减少Fov（视场角）

按Page Down键可以增加Fov（视场角）

按Home键，恢复Fov（视场角），space engine 0.990默认的是45°

左手一直按住Shift键，右手一直按住鼠标左键，拖动鼠标向前滑动或者向右滑动，也可以减少和增加Fov（视场角）

2.3 天体基本操作

2.31 右键菜单

选中任何一个天体，在这个天体上鼠标右击，会弹出一个选择菜单。

info 信息：在天体上右击选择这个，会弹出Wiki信息表，在上面会显示这个天体的相关数据信息.点击上面的

标签会显示相关的信息。

Parent 父体：

在恒星上右击选择这个选项，左上角会出现系统质心数据信息。

在行星右击选择这个，左上角会出现母星（恒星）的数据信息。

在行星右击选择这个，左上角会出现其对应的行星的数据信息。

Browse Planetary system 行星系统浏览器：选择这个，左上角会打开。

在行星和卫星上右击选择这个，左上角会弹出恒星和行星浏览器，显示行星和卫星。

在恒星上右击选择，左上角会弹出恒星和行星浏览器，恒星，行星和卫星。

Add to locations 添加位置（坐标），右击选择这个，会把天体添加到坐标浏览器里。

Hide/show object 隐藏/显示物体（天体），右击选择这个，会暂时在原位置隐藏被选择的物体（天体），在原位置继续选择这个，会显示被隐藏的物体（天体）。

Go to object  前往物体(天体)，选择这个，会到达被选择的物体(天体)跟前。

在Space Engine里面任何一个亮点都可能是天体，选择要去的天体，右击选择Go to object可以达到这个天体跟前，也可以按G键到达。

Land on object 登陆物体（天体），点击这个，会自动登陆物体（天体）表面。

Center object  物体（天体）中心，选择这个，物体（天体）会处于屏幕中心。

Follow object 跟随物体物体（天体） ，选择这个，在物体（天体）上鼠标一直按住鼠标右键，左右或者向下拖动，相机会围绕物体物体（天体）旋转，这是在默认的自由模式下的。

按数字键2，进入飞船模式，这时候Follow object 跟随物体物体（天体），在物体（天体）上鼠标一直按住鼠标右键，向右拖动鼠标，这时候相机就会向左旋转运动。

数字键2和数字键3效果一样

Rotate with object 旋转物体（天体），这个和Follow object的有点像

Track object 跟踪物体（天体），右击选择这个，会锁定选择的物体（天体），无法移动

2.32 搜索天体

搜索恒星，行星，黑洞，星系，星云等天体有4种方法：

第1种：使用天体编号搜索

查询维基或者天体数据库获得天体的编号，点击Space Engine界面左边的第7个像放大镜的图标，打开Find  object 查找物体（天体）搜索器，在搜素框里输入天体编号，选中搜到的天体名字，点击GO to（前往）到达，比如输入Sun

天体编号有空格的要输入空格，比如输入M 87

有的天体有多个天体编号，如果第一个输入搜不到，尝试输入后面的编号搜索。

第2种：使用坐标搜索

Space Engine里面的程序天体会有一串字母和数字组成的坐标，当别人分享这个坐标的时候，打开Find  object 查找物体（天体）搜索器，输入坐标可以搜到相关天体，坐标有空格的要加上空格。

Space Engine各版本的坐标不通用

第3种：按照名字搜索

如果创建了天体，要输入脚本里面创建的天体名字，通常是英文或者英文和数字组合。

第4种：按范围和条件搜索

按Shift+F3键或者点击主界面左边菜单栏上的第6个图标，打开Star browser（恒星，行星浏览器）

在Search radius(搜索半径)里输入100

点击Filter settings(过滤设置)，打开Filte settings对话框进行相关设置，设置往后，点OK，

然后点Star browser里的Search(搜索)，可以搜到设置的范围内符合条件的天体，搜到的相关天体信息会出现在Star browser列表里，点击任何一个天体坐标，点Go to可以到达

这儿可以搜到存在于Space Engine里的任何类型的天体。

另一种搜索黑洞的方法是，选中任何一个星系，按2次Ctrl+G，就可以到达这个星系中心的黑洞。对于星团也可以尝试这样操作，可以到达黑洞.

2.33 保存天体（坐标）

保存天体有2种方法

第1种：以当前位置保存天体：

选中当前要保存的天体，在这个天体上鼠标右击，选择Add to locations 添加位置（坐标）或者按主界面左边菜单里面的第10个像小旗的图标，打开Locations位置(坐标)浏览器。

点Add(添加)，选择This location（当前位置），就会把选中的天体以当前位置添加到了位置(坐标)浏览器里，这时候上面会多出一个图标。

下次要想查看这个天体，只要选中图标，点Go to（前往）就可以到达。

第2种，保存别人分享的坐标找到的天体：

打开Locations位置(坐标)浏览器，点Add(添加)，选择Paste  URL from clipboard(粘贴剪切板中的URL)，把URL粘贴到这儿。

Space Engine 0.990里的URL是Space Engine一种新的分享坐标方式

2.34 重命名天体

在Locations位置(坐标)浏览器中，要重命名哪个天体，就选中这个天体的图标，点Rename(重命名)

2.35 删除天体

在Locations位置(坐标)浏览器中，要删除哪个天体，就选中这个天体的图标，点Delete(删除)

2.36 分享天体坐标

分享天体坐标有2种方法

第1种，通用坐标分享

真实天体和程序天体都是按照这个方法分享坐标（位置）。

选中要分享坐标的天体，在这个天体上鼠标右击或者点击主界面第2个图标，打开Wiki 信息框，点击Edit(编辑)，然后在Name右边的长方框里点击下，激活光标。按Ctrl+C，就可以复制当前天体坐标到剪切板，分享给其他玩家就可以了。

对于真实天体，通常有多个编号（名字），搜索第1个或者第2个编号（名字）就可以搜到。

对于程序天体，只有由字母，数字，符号组成的一个坐标。

第2种，从Locations 位置(坐标) 浏览器分享

从这儿分享的URL坐标就像网页网址一样的代码。

打开Locations 位置(坐标) 浏览器，选中要分享的天体的图标，点Share( 分享)，选择Copy URL to clipboard(复制URL到剪贴板)

2.34 使用Solor system browser(太阳系浏览器)查看系统内天体（查看恒星，行星，卫星，黑洞，小行等等天体）

Space Engine里的Solor system browser(太阳系浏览器)，用于查看恒星，行星，卫星，黑洞，小行等等天所在的系统内全部天体。

选中一个天体（非星云，星系，行管），按F2或者点space engine主界面第4个像齿轮的图标，左上角会弹出Solor system browser(太阳系浏览器)，再次按F2会关闭Solor system browser(太阳系浏览器)。

点击里面选择一个天体的图标，左上角会显示选择的当前天体信息。

选中里面的任何一个天体图标，按G键可以前往

在恒星或者行星图标上鼠标右击，会进入下一页，显示恒星的行星或者行星的卫星。这时候，在恒星或者行星上右击，会返回上一页，对于双行星或者双恒星系统的，点击左边的质心图标会返回上一页

点击Solor system browser(太阳系浏览器)左边的齿轮图标，打开显示设置，可以设置以下：

Dwarf Planets，选中这个，在Solor system browser(太阳系浏览器)里会显示矮行星图标，取消后，不会显示

Gas giant minor moos，选中这个，在Solor system browser(太阳系浏览器)里会显示气态行星的微小卫星图标，取消后，不会显示。

ReaL sizes 真实大小，选中这个，在Solor system browser(太阳系浏览器)里会显示天体真实大小

View angle 查看角度

再次点击这个齿轮图标，会关闭显示设置

2.35 使用Solor system chart(太阳系图表)查看一个系统里的天体比较

选中任何一个天体（非星云，星系，星团），点击主界面左边的第5个图标，打开Solor system chart(太阳系图表)

在这个图表里面会显示当前天体所在的系统全部的天体，并且按照默认的方式排列，在右边可以选择以什么方式排列天体，选择显示和不显示指定的类型的天体。

默认的一些天体不显示，可以在Show（显示）列表里打勾，就显示了。

按W键，前进，按S键，后退，

按A键，向左移动，按D键，向右移动

用鼠标也可以操作，按住鼠标左键，可以上下移动

鼠标滑轮向前或者向后控制前进和后退

通过键盘和鼠标的合作，更容易查看当前系统内的各个天体。

在这个Solor system chart(太阳系图表)界面里，如果想去哪个天体，在这个天体上鼠标右击，选择GO to object (前往)

2.36 开启轨道线

按字母O键或者点击space engine主界面右边的第3个图标可以开启轨道线

再次按字母O键或者点击space engine主界面右边的第3个图标可以开启轨道线

打开点击General（常规设置），点击Visual style(视觉样式)，在Orbit maks(轨道标记)那儿可以设置轨道线的曲线

点击space engine主界面右边的第19个图标，打开Filter objects过滤物体(天体)，右边Orbits(轨道)下面的选项可以选择显示和不显示一些天体的轨道线，以及选择显示和不显示天体的标签。

2.37 时间控制菜单

在space engine主界面左下角是时间控制菜单，按空格键是开启时间，再次按空格键是关闭时间。

如果是新启动Space Engine，按空格键，就是开启了默认的速度，默认的速度是非常慢的，除了当前是一些高速运动的天体，会运动很快。

如果当前已经加速或者减速到一个速度了，按空格暂停后，再次按空格键是恢复前面设定的速度。

开启时间，适当加速后，会看到恒星，行星，卫星，小行星，彗星等天体的运动。如果开启了轨道线，可以看到天体按照轨道线运动。

星云，星系，星团，不会有运动。

2.38 打开universe map(宇宙地图)

点击space engine主界面左边第3个图标，打开universe map(宇宙地图)窗口

把链接复制到浏览器地址栏上，按回车键就可以看了。
