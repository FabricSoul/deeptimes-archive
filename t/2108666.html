凌晨2点多的新闻42号：

X3: Terran Conflict Update 3.1

X3：地球人冲突 升级档3.1

This is an update for version 3.0 onwards.

这是一个只能用于3.0 到3.1的升级档

This update brings a large number of fixes and improvements to many parts of the game.

这个升级在游戏的各方面都有修复与改善

Version 3.1 features:

3.1版本特点：

    * Fixed occasional freeze issue in the Balance of Power mission

       修复了Balance of Power 任务（3.0任务）中时不时“冻住”的问题（没遇到过，是任务做了一半不给接下半段，要等老久才给接吧）。

    * Fixed case where the Balance of Power achievement may not unlock

       修复了Balance of Power 任务成就可能不解开的问题（Steam 成就吧）

    * Some minor mission fixes

       一些任务小修复

    * Fixed some map issues (for new games only)

       修复了一些地图问题（需要新开游戏……）

    * Several text fixes

       一些文字修复

    * Fixed blank messages when object names are coloured by mods

       修复了Mod中彩色字体名称的物体发来的消息是空白

    * Fixed several passenger transport missions which could break in some circumstances

       修复了一些送人任务无法完成

    * Added Cargo Lifesupport System to Boron Angel

       给Boron 的Angle 加了内置货仓维生装置

Additional information about X3: Terran Conflict 3.1 can be found in the 3.1 News Post and change log.

另外的信息可以在新闻和变化表中找到。

Attention: This update is not compatible with the Steam version of X3TC! The Steam client will automatically update your game files.

注意：这个版本不适合Steam版！Steam版会自动升级。

Once downloaded, simply run the .exe and follow the instructions.

下载好后，运行EXE并遵循导引

Available Languages: English, French, German, Italian, Spanish, Polish and Russian.

语言：英语，法语，德语，意大利语，西班牙语，波兰语和俄语。（能看懂这段的就不用看我翻译的中文了）

下载地址：http://www.egosoft.com/download/x3tc/patches_en.php

引用官方人员的一段话：In perhaps the final update to X3: Terran Conflict as our efforts fall full-square behind TNBT, we introduce some further bug-fixing, particularly for the Balance of Power missions of version 3.0. We had hoped to introduce these fixes sooner, but as you can imagine, we've been pretty busy on other projects recently!

    在我们全身心投入TNBT（The Next Big Thing，下一个游戏，不再是X了）时这可能是X3TC的最后一个升级了。我们带来了更多的漏洞修复，特别是Balance of Power任务。我们本想早些发布，但是你能想象到，我们现在多么忙于别的计划！
