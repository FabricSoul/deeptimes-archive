宇宙模拟器Space Engine 0.990教程3:截图和视频录制，VR截图和录制VR视频，导出行星纹理

提示：鼠标移动到图标上，会显示当前图标代表的功能。

对于视频录制，我推荐的还是Space Engine自带的视频录制工具，录制出来的视频比较平滑。

3.1 截图

按F11键截图，不会带界面信息

按Crl+F11键截图，会带界面信息

点击界面右下键的相机图标，可以快速打开截图所在的screenshots文件夹

3.2 录制视频

3.21 用录制AVI视频方式录制视频

Space Engine目前录制的视频格式只有AVI格式，使用这种录制方式，要受到Space Engine 自带编码器限制，录制的视频时长不能超过30秒。

如果是使用第三方录制软件录制，不用考虑Space Engine对录制视频时长的限制。

按F9打开Video capture settings(视频编辑)

选中Rerord in AVI(保存为AVI)，点击Configure codec(配置编码解码器)，点击“压缩程序”，往下划选择“全帧（非压缩）”，在Engine frame rate和Video frame rate后面都改成30

在With interface(包含界面)前面的方框打上勾，会把界面的都录制下来，取消，不会录制界面。

点“录制”按钮就开始录制视频，在Time后面会显示当前已经录制了多少时间。

受到Space Engine 自带编码器限制，录制不要超过30秒时间，在30秒内的时间内，点“停止”按钮，视频就录制好了。

点space engine主界面右下键的相机图标，打开screenshots文件夹，在这个文件夹里面的capture文件夹里，就是录制好的视频。

3.22 用图片序列帧方式录制视频

按F9打开Video capture settings(视频编辑)，选中Record in screenshots（保存为截图）

默认的截图格式是jpg,点击可以修改成其他格式的。

如果改成png格式的话，会激活格式选择下面的Save transparency（保存为带透明通道的图片），这个属于Pro DLC的功能。

设置完后，点“录制”按钮就开始录制视频，在Time后面会显示当前已经录制了多少时间。

受到自带编码器限制，录制不要超过30秒时间，在30秒内的时间内，点“暂停”按钮。视频就录制好了。

点space engine主界面右下键的相机图标，打开screenshots文件夹，在这个文件夹里面的capture文件夹里，就是录制好的视频。

使用这种录制方法，不受Space Engine的限制，可以录制很长时间。

点“停止”按钮，就录制好了。点space engine主界面右下键的相机图标，打开screenshots文件夹，在这个文件夹里面的capture文件夹里，是一张张连续的图片，这就是录制好的视频序列图片，导入Premiere等视频编辑软件里面可以合成视频。

3.3 Camera path editor 摄像机路径编辑器

这个功能属于pro DLC用户的，使用这个工具，可以在平常观光浏览的状态下记录浏览过程（路径），然后使用“重现并录制”录制视频。使用这个工具录制视频，如果是直接使用AVI视频录制的方式，还是受到Space engine自身限制，需要控制录制的时间在30秒内。如果是使用录制图片序列帧方式录制，不受这个限制，可以录制很长时间。

点击“重现”按钮，可以用第三方录制软件录制，不用考虑Space Engine对录制视频时长的限制。

按F9打开Video capture settings(视频编辑)，点击Edit Camera path(编辑摄像机路径)打开Camera path editor （摄像机路径编辑器）。

取消在Display path curve(显示路径曲线)前面的勾，可以取消，在录制视频的时候，不会把路径曲线路径进去。

点击Camera path editor （摄像机路径编辑器)上的“录制”按钮，就开始记录浏览路径了，按照平时的观光浏览就可以了。在记录浏览路径的过程中,会在Points(记录点)后面出现记录点数，下面会出现红色的Recording...(录制中)，点击“暂停”按钮，就停止了录制浏览路径。

点击“重现”按钮，Space engine会把之前浏览的过程重新播放一遍，在这个重现的过程中，可以使用第三方视频录制软件录制下来。

点击“重现并录制”按钮，会重新播放浏览过程的同时，录制视频，在Video capture settings(视频编辑)里面会看到录制时候的参数，录制完后，自动会停止。

如果你要保存录制的路径，当路径录制好后，点击Save(保存)，可以打开保存窗口选择一个位置，重起一个名字，名字必须是英文的。如果想导入路径，点击Load(加载)，选择要导入的路径文件，导入的路径文件扩展名是.ssp

3.4 Space Engine全景VR360°截图和 录制VR360°视频

点space engine主界面第11个像显示器的图标，打开Settings(设置)窗口，在在Display(显示)里选择Cylindrical(圆柱形模式)，在这个模式下，截图和录制视频都是全景VR 360度的，截图和录制视频方法跟在普通观光模式下是一样的。

如果要回到正常的观光模式，同样打开在Display(显示)，点Perspective(观光浏览)

3.5 Space Engine连接VR头盔

电脑连接好VR头盔，打开Settings(设置)窗口，在在Display(显示)里选择Virtual reality headset(VR头盔)，右上角会有一些参数设置。

如果要回到正常的观光模式，同样打开在Display(显示)，点Perspective(观光浏览)

3.6 导出行星纹理

选中一个行星，按Shift+F2键打开编辑器，选择Export textures(导出纹理)，打开Export Planet textures(导出行星纹理)窗口，在Resolution(分辨率)里输入一定的数字，默认是1024，根据自己的显卡显存，可以改成2048，甚至更大的分辨率。在Image format( 图片格式)中选择要导出的纹理图片格式。点Export(导出)，当界面提示Done(完成)，打开Space Engine安装目录里的export文件夹，就可以看到导出的行星纹理图片。

重复导出会替换上次导出的纹理，假如，你第1次按照1024分辨率的纹理，第2次导出2048分辨率的纹理，会替换export文件夹里面第2次导出的纹理。如果，你要保存每次导出的纹理，就需要每次把导出的纹理移动到其他文件夹里。

3.7 Cubemap（立方贴图）

点击space engine主界面左边第11个图标，进Display（显示）设置菜单，选中 Cubemap（立方贴图）,界面会进入 Cubemap（立方贴图）显示模式，右边可以设置一些分辨率参数和模板。

把链接复制到浏览器地址栏上，按回车键就可以看了。
