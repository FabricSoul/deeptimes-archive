什么是CAG？自己也琢磨了很久，为了避免新人重蹈覆辙，在这里做下详细的说明：

CAG是Commercial Agent的缩写（高手一般都只用缩写的说 ），在Bouns Package内有相应脚本。具体功能如下：

CAG为指定Homebase执行购买、销售任务

CAG在区域内工厂、贸易中心、设备中心间执行最低价买进、最高价卖出。

CAG在在复合工厂中间产品生产不足时或产出过多时提供买进、卖出服务。

CAG将其Homebase内所有产品作为交易对象，但玩家可对其中某些产品予以设定不参与交易。

CAG遵守玩家设定的购买、销售规则，并可生成一份购买、销售明细列表。

CAG将会与其星区内其他CAG间相互协调。

CAG将会监视器Homebase资金情况，并在现金不足时卖出产品。

CAG将会根据最佳交易该表交易路线。

CAG会使用Jumpdrive并维修自己。

CAG会与其客户保持联系以便了解其星区内价格以及其他及时信息。CAG不是战斗单位，将会避免前往被报告有敌人出现的星区。（似乎只有CAG之间可以报告敌人出现情况，卫星或玩家似乎不行）

CAG的具体能力将随其训练等级逐步提升，并接受更多任务。当然，培训课程是可以手动终止的。（没人会想这么做吧？）

如何，感觉CAG够强大么？想要么？

那就继续往下看：

当然，以上所涉及到的种种功能是随着飞行员等级逐步提升才能掌握的……  

具体等级对应能力说明如下：

Apprentice        仅采购资源，不领工资

Courier             采购资源并售出产品，可被指派给Equipment Docks 与 Trade Stations。工资20 Cr每Mizura（X宇宙时间）

Supplier            会与其星区内其他CAG协调，会购买无人战斗机并对船速、转向能力升级。工资25 Cr每Mizura（X宇宙时间）

Trader                可购买并销售复合工厂中间产品，会使用Jumpdrive。工资30 Cr每Mizura（X宇宙时间）

Dealer                开始关注Homebase的资金情况，并在资金不足时出售其产品。同时关注交易商之间的信息交易，并避免前往被报告有敌人出没的星区。工资35 Cr每Mizura（X宇宙时间）

Major Dealer        在飞行过程中持续关注价格信息，并会在必要时更改航线。在飞机受损时会自动前往船厂修复。工资40 Cr每Mizura（X宇宙时间）

明白了吧？CAG可是要工资的哦~

如何才能使用CAG？

一架灰机~设定好基地并安装如下软件：

Navigation Software MK1, Trade Command Software MK1 and MK2（必装）

Fight Command Software MK1（在当前星区自动购买无人战机，并升级速度、转向。并可能使用Jumpdrive进入非安全区域）

Fight Command Software MK2（在当前星区自动购买蚊子防御系统）

Jumpdrive（达到一定等级后方可使用）

全部安装完成后

打开飞机的命令菜单，在Trade命令内选择Start Commercial Representation即可进入CAG设定

做一个很小的补充：全局命令还可以通过设置快捷键调出。全局命令的分配可以过滤，比如你设置了每艘飞船的飞船编号，想要改变所有飞船的某一项设置但又不想改变飞船编号时可以造过滤目录中关闭飞船编号一项，这样将全局命令分配给所有飞船时就不会改变每一艘飞船的编号了。
