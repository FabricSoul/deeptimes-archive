X3 Reunion : 重聚作为一个成熟的老游戏，其脚本非常多，也非常完善，建议新手了解脚本从此开始。

以下所有脚本均经本人测试不冲突，放心使用，但不在下列范围的可能有脚本冲突。

这些脚本具体说明都可以去下载电子书（120个脚本电子书完善版）

http://www.rayfile.com/files/cb4b404c-7591-11de-8dda-0014221b798a/

脚本下载可以直接用电子书里的链接，如果你需要中文版或者电子书里没有的可以去

http://coolook.ys168.com/

当然你也可以访问作者的主页下载一些链接失效或更新的版本

http://www.xai-corp.net/

http://cycrow.thexuniverse.us/

本人将脚本分为两大类

一、必装脚本（该类脚本不破坏游戏平衡，添加新元素或简化部分操作）

1、Cycrow的Guilds (公會)/Military (軍事) 脚本系列

推荐指数：★★★★★

极大的丰富了游戏的可玩性，让你不用为钱而挂机，各种公会任务可以让打发漫长的UT赚钱时间。

暂时发现Taxi脚本有冲突。

2、BBS Extras (BBS買賣廣告)

推荐指数：★★★★★

丰富了BBS内容，方便玩家进行某些事，极大的改观X3原有一些死板的规定，使X3更贴近现实，更有趣，比如跳跃引擎原版只有两个地方卖，现在你可以通过BBS邮购，当然价格高多了。

如果你喜欢X3原有的那些死板规定我也没话说。

3、Pilot AI

推荐指数：★★★★★

极大的丰富玩家战术指令，开启战队指挥等的玩法，功能太多，太强大，可以说提升了X3耐玩度一个档次。

使用心得：

http://bbs.deeptimes.org/forum.php?mod=viewthread&tid=30153&extra=page%3D1

http://bbs.deeptimes.org/forum.php?mod=viewthread&tid=30235&extra=page%3D1

http://bbs.deeptimes.org/forum.php?mod=viewthread&tid=30233&extra=page%3D1

4、Ashleys Factories

推荐指数：★★★★★

玩家后期工厂集群必备，提高游戏速度，简化工厂搭建难度等等。

初期还是搞个小集群吧，练习下盖工厂。

5、X³ All In One Bonus Package

推荐指数：★★★★★

官方脚本补丁，简化丰富部分指令，说明看我的电子书。

装了Pilot AI就不要装GHK了，Pilot AI包含了GHK功能。

6、Turret Commands（炮台指令，去下http://coolook.ys168.com/里的，我的电子书中那个是另一种）

推荐指数：★★★★★

非常有用的脚本，不同武器对不同的船，但是换武器太麻烦，而且当你使用有炮塔的大船时，更痛苦，不仅要换武器，还要换炮塔指令，这个脚本可以保存你的5种炮塔配置和指令，用于快速切换。（就是使用上手比较麻烦，需要研究下才行。）

武器能量低的M3、M4、M5之流，就没必要了，把每个炮台都装满，用游戏自带的武器分组吧。

7、Weapons Changer

推荐指数：★★★★

Turret Commands的简化版，这个可以保存3种武器配置，但无炮塔指令。

建议与上面的脚本配合使用，Turret Commands配置大船，Weapons Changer配置小船。

本人战斗习惯是大船先控制下，打的差不多了，就切换小船，领导队伍战斗。同样可以广播到整个星区所有船上。

说明里的OOS模式就是武器更换界面。

8、Satalite Early Warning Network (人造衛星預警網絡)

推荐指数：★★★★

优点：可以语音提示你那里有敌情，如果你打开字幕了的话，更直观。BBS里可以买卫星维护服务，价格便宜，卫星损毁后，会自动补个卫星到那里。不用总放卫星了。

缺点：语音提示较多，有点烦，部分没有空间站的星区就没有卫星维护服务。

如果你不需要卫星维护服务，又讨厌这个语言提示，建议你就用Pilot AI的COMIA监控网络就可以，随时调用监控网状态，就可以查看哪里有敌军了。

9、Move To Coordinates (座標式移動指令)

推荐指数：★★★★★

小脚本，大作用，移动到指定坐标，

格式有两种：按先后循序对于xyz

100,100,100

回车后移动到x：100m，y：100m，z：100m

100k,100k,100k

回车后移动到x：100km，y：100km，z：100km

10、AutoDrone launcher plugin (微型戰鬥機插件)

推荐指数：★★★★

自动释放无人战机，感觉用处不是太大，给TS装上还可以。

11、AI Spy Satellites (AI間諜衛星)

推荐指数：★★★★★

奶奶的太有用了，可以在敌对星区放卫星而又不用担心被摧毁，而且他是特别的卫星，不会被其他脚本所利用，可以设置开关，一般情况下我都是让他关闭的，减少被敌发现几率，要查看时再激活就可以了，使用非常方便。

12、Treasurehunt (財寶獵人)

推荐指数：★★★★★

小插件，随机增加各种无人船，提升游戏乐趣。

13、Freight Transporter Upgrade (太空貨櫃回收套件)

推荐指数：★★★★★

战斗完了，太空里有很多的货物，不捡的话实在有些败家，但是捡起来实在麻烦，这个插件可以让你快速收集你飞船周围4KM范围内的所有货物，仅玩家控制船有效。

用来捡导弹之类刚好，货船掉的还是让货船来捡吧，一般你用M3的话，货船的东西都很多，仓不够捡不了的。

14、Race Response Fleets (種族應變部隊)（可能与Pilot AI有冲突，但我未发现问题。）

推荐指数：★★★★

电脑不是那么好虐了，打其一必定牵动其全身，即使你误伤了对方的空间站，也要赔款，甚至赔款晚了点，赔款还要加倍，你们自己小心。

建议如果装了这个，就必装Ship Jamming Device (訊號干擾裝置)，阻止他呼叫援军，这样也可以杀人越货，但是你要小心才行。

15、Advanced Tractor Beam (高級牽引射線)

推荐指数：★★★★

工厂位置不好？放的位置不满意？可以改的，这个我想也是应该的，毕竟这玩意价格不菲。

16、Navigation Override System Mk1 (NOS) (飛船控制程式 Mk1)

推荐指数：★★★★

聊胜于无吧，这个脚本反正我用着不太爽，他看见敌人，又不语音提示，最后被炸了才告诉我，他曾经尝试过逃跑。

17、EMP

这个不多说，相当于脚本语言库，很多脚本需要这个为基础。必装。

二、选用型脚本（这类脚本对游戏有轻微的平衡破坏，但是相对于其带来的游戏乐趣，我是都装了的，太逆天的脚本或者用处不太大的脚本我没有放这里。）

1、Nividium Mines (Nividium礦源)

推荐指数：★★★

我宁愿相信这个是X3的一个BUG，Nividium矿太少，而且分布极端。有点作弊嫌疑。

2、Salvage Claim Software (飛船拖運裝置)

推荐指数：★★★★

大名鼎鼎的抓船工具，建议初期还是老老实实下飞船去抓吧，体验下乐趣，但是这个抓船这个东西完全就是个体力活，体验下太空漫步就可以了，不用总体验。

绝对不要用这个抓M6以上的大船，虽说可以抓，但是那就严重破坏平衡了。自觉啊同志们！

3、Advanced Jumpdrive (高級超空間跳躍裝置)

推荐指数：★★★★

个人建议座机M3用用就好，M3的仓实在是小，带不了太多东西，我的座机LX又有两套武器，实在是没空间搞电池，而且还要搜集宇航员和武器弹药。

还是靠大家自觉，如果舰队大规模配，就没意思了，M1、M2也不要配，仓大，让补给舰有点事做吧。当然你想大规模配，1000万的价格也不允许。

4、BBS Overtune Ships (改裝飛船服務)

推荐指数：★★★★

轻度破坏游戏平衡，突出玩家优势，对于不喜欢同种船AI和玩家一样性能的玩家推荐使用，可以牺牲船的某些性能提升最大速度。

5、Deathmatch Arena (死亡競技場)

推荐指数：★★★★★

提升游戏乐趣无限，奖品多多，但是奖品都有些逆天，不过得之也是不易。

6、Heavy Assault Capture Tech (俘虜重型艦)

推荐指数：★★★★★

稍微有点作弊，本身这个设备就要高达近7000万才能买到，再加上点钱都可以买新船了。我主要是看中了使用的条件很真实，派遣部队到对方大舰抢船本来就是存在的。乐趣在于可以抢些搞不到的船，同时对于可以弄到的船，如何控制火力，将对方的盾保持在20%以下，又不伤HULL，还要保持5KM距离，就是个技术活了。

7、BBS Player HQ Blueprint Offers (飛船藍圖郵購服務)

推荐指数：★★★★

相信很多玩家都是完美主义者，希望HQ能生产所有的船，但是那需要非常大的工作量，这个东西就是解决方案。

个人建议不要买那些不好搞的船的图纸，那降低了不少乐趣。

8、Reveal Pandora Crate Positions (顯示潘朵拉之盒位置)

推荐指数：★★★

相信大家玩了很长时间后，可能找到了几个，但是一共10个还是不好找的，总不至于为了这玩意刷bbs吧。这个脚本帮你找全他们。

建议找到8个以上再用这个脚本。
